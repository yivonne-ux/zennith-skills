## Human Approval Before Upload (2026-03-22)

User said: "All artwork needs to be approved by human before upload ya."

**Why:** Quality control. AI-generated creatives can have text errors, wrong food, wrong mood, cropped content. Human eye catches what automation misses.

**How to apply:**
1. Generate artwork → save to `06_exports/campaigns/[campaign]/`
2. Open folder for review → tell user "review these"
3. WAIT for explicit approval ("approved", "go", "upload these")
4. ONLY THEN upload to Meta and set ACTIVE

**NEVER:**
- Generate and upload in one step
- Auto-activate without human seeing the artwork
- Assume approval — always ask

This applies to ALL brands, ALL output types, ALL campaigns.
