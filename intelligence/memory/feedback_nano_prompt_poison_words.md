NEVER use these words in NANO prompts for food/bento/labeled posts:
- "sparkle dust particles", "sparkle dust", "sparkle energy"
- "glitter", "glitter particles", "pink glitter dust"
- "candlelit glow", "candlelit atmosphere", "candlelit warmth"
- "bokeh", "bokeh light circles"
- "dreamy pink glow", "dreamy"
- "rose gold" (as label color — bleeds onto food)
- "atmospheric", "atmosphere"
- "golden warm light from the side"
- "warm candlelit"

These cause NANO to: generate BLACK/DARK backgrounds, add gems/bling ON food, create moody/dark scenes.

**Why:** Batch2 v6 regression (March 29 2026). B2-13 and B2-14 came out with completely black backgrounds. B2-11 had glitter bg. The proven EXTRA-labeled posts (42-posts batch) used NONE of these words.

**How to apply:** For food posts, use ONLY: "warm blush pink background. Clean editorial feel." — that's it. No atmospheric modifiers. The proven formula is dead simple. For label text color: "dark grey/brown text on subtle white label boxes" — never "rose gold".

Also: do NOT use aesthetic refs (sparkle Pinterest pins) as Image 1 for food posts. Image 1 = the enhanced food photo itself. NANO should only add text/labels, not restyle the entire scene.
