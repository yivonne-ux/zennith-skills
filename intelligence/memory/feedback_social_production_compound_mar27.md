## SOCIAL MEDIA PRODUCTION — COMPOUND LEARNING (March 27, 2026)

### SESSION SUMMARY
- Brand: Pinxin Vegan Cuisine

- Rounds: 5 illustration calibration rounds + 2 fix rounds + 1 carousel variety round

---

### ✅ YES — What Works

**Illustration Pipeline (v5 — proven):**
1. MAX 2 reference images (character refs only) — less dilution = better style match
2. Food described in TEXT only for illustration posts — no cutout images (prevents style mismatch)
3. Vegan/dietary gate at END of prompt — NANO reads end more strongly
4. Full brand palette (all 7 PX colors) — not reduced to 2 = richer, more branded
5. Character proportion instructions specific ("1.3x eyes, rosy cheeks + nose, catchlight")
6. "This is the #1 most important instruction" = priority signal works
7. Micro-detail instructions work (labeled jars, recipe notebooks, scattered ingredients)

**Food Hero Pipeline (proven):**
8. Mood ref as Image 1 + food cutout as Image 2 = reliable
9. Different mood refs per dish = good color variety (sage, burgundy, dark, green-gold)
10. Signature marker descriptions in prompt = food accuracy

**Carousel (proven after fixes):**
11. W1-04-S1 style (typography + gold line-art on PX Green) = best hook slide approach
12. Using own proven output as reference = better than prompting from scratch
13. Different dish + different vessel per slide = good swipe experience
14. Color flow across slides (cream → green → burgundy → dark) = engaging
15. Recipe carousels: same dish, different PREPARATIONS (noodle, dry, hotpot)
16. Educational carousels: different dish per slide showing RANGE

**Content Planning:**
17. 10 content categories, not just 2 = variety
18. ACCA funnel mapping = strategic
19. Grid rhythm BOLD → PHOTO → CLEAN = visual flow
20. High-shareability formats (myth-bust, this-or-that, recipe carousel) = engagement

**Comics (proven):**
21. 4-panel format with speech bubbles works well in NANO
22. Relatable scenarios (office lunch, mum's tupperware) = shareable
23. Refined linework instructions improve quality (but NANO has ceiling)
24. Full palette in clothing/environment = branded feel

### ❌ NO — What Fails

**Illustration:**
1. ❌ More than 2 reference images = NANO dilutes, generic output
2. ❌ Realistic food cutouts in illustrated scenes = style mismatch
3. ❌ Forced brand elements (ginkgo frame, botanical border) = "too overwhelming"
4. ❌ Anime/manga reference for Chinese food brand = cultural mismatch
5. ❌ Reducing palette to 2 colors = flat, unbranded
6. ❌ Vegan gate at START of prompt = buried, ignored by NANO
7. ❌ Product names with trigger words ("fish") = NANO draws them literally

**Food Hero:**
8. ❌ AI-generated food (not from cutout) = non-PX dishes, vegan violations
9. ❌ Same food cutout for every carousel slide = monotonous
10. ❌ Curry in a teacup = weird vessel choice. Use proper dinnerware.

**Carousel:**
11. ❌ Same dish on same plate across all slides = boring swipe, rejected
12. ❌ Food photo mixed with rough illustrations on same slide = clashing styles
13. ❌ Duplicate CTA text = layout glitch
14. ❌ Calligraphy/text bleed from NANO training data = needs clean prompt

**Content Planning:**
15. ❌ Only 2 post types (food hero + illustration) = no variation, rejected
16. ❌ Building campaign around non-core festival (Hari Raya for Chinese brand) = misfit
17. ❌ Assuming commerce events (4.4 Sale) apply to all markets = unchecked
18. ❌ No calendar verification = wrong festival dates

**Process:**
19. ❌ Generating from scratch without reference = generic AI output
20. ❌ Not auditing carousel as a SWIPE EXPERIENCE = missing variety issues
21. ❌ Fixing only flagged issues instead of auditing ALL similar posts = systemic issues missed

### 🔧 PRODUCTION RULES (Enforced — All Brands)

| Rule | Detail |
|------|--------|
| **Reference first** | EVERY generation starts from a reference image. Never from imagination. |
| **Max 2 ref images** | For illustration. More = dilution. |
| **Food = text only** | In illustration posts. No cutout images. |
| **Dietary gate last** | End of prompt = strongest signal. |
| **Full palette** | Use ALL brand colors. Never reduce. |
| **Carousel variety** | Different dish + vessel + color per slide. Audit as swipe experience. |
| **Illustration-led hooks** | Typography + gold line-art on brand bg. No food photo on hook slides. |
| **Own output as ref** | Approved outputs become references for future generations. |
| **Calendar check** | Verify festival fit before building campaigns. |
| **Audit ALL similar** | When one post has an issue, check all posts of same type. |
