## CAROUSEL VISUAL VARIETY — UNIVERSAL RULE

### The Problem
PX carousel slides all showed the same rendang on the same white scalloped plate. User rejected as "boring, no variation." Even with different text/layout per slide, the visual content was identical.

### Rules

1. **Each slide must have a visually DIFFERENT hero image.** Not the same food. Not the same plate. Not the same angle.

2. **For recipe carousels** (same dish, different preparations):
   - Same base ingredient but presented DIFFERENTLY per slide
   - Different vessel per slide (bowl → flat plate → clay pot → hotpot)
   - Different accompaniments (with noodles → with rice → as soup → as hotpot base)
   - The dish should look TRANSFORMED in each slide

3. **For educational carousels** (myth-busting, facts, benefits):
   - DIFFERENT dish per slide to demonstrate range
   - Rotate through brand's product catalog
   - Each slide = different color mood (green curry = green, vinegar = dark, sambal = red)
   - Hook slide (S1) should show MULTIPLE small dishes to signal variety

4. **For ingredient spotlight carousels:**
   - S1: Hook (illustrated/styled ingredient)
   - Middle slides: Show the ingredient IN different dishes (soup, curry, stir-fry)
   - Mix raw ingredient shots with finished dish shots
   - CTA slide: hero dish

5. **Color flow across slides:** Each slide should have a noticeably different dominant color when swiping. E.g.: cream → green → burgundy → dark. Never two adjacent slides with same color world.

6. **Use DIFFERENT food cutouts** per slide. Never pass the same cutout to every NANO call in a carousel batch.

### 3 Carousel Visual Approaches (mix them)

1. **Food photo** — mood ref + food cutout. Editorial photography. Best for: dish hero slides, CTA slides.
2. **Illustration-led** — PX Green bg + gold line-art food illustrations + bold typography. Best for: hook slides, fact slides. (Proven in PX-W1-04-S1, PX-W1-10-S1 v2.)
3. **Mixed** — illustration accents + real food photo in same slide. Or alternate between photo and illustration slides. Creates the richest swipe experience.

**W1-04-S1 is the reference** for illustration-led carousel slides. PX Green + gold botanical/food line art + massive CN typography.

### Anti-patterns
- Curry in a cup = weird. Use proper vessels (bowls, plates, clay pots, hotpots).
- Same food cutout for every slide = visually identical batch = rejected.
- All-photo carousel with no visual variation = boring swipe.

### Pre-flight Before Carousel Production
- [ ] Each slide uses a different food cutout image
- [ ] Each slide has a different vessel/plating
- [ ] Color palette shifts between slides (noticeable when swiping)
- [ ] No two adjacent slides look visually similar
- [ ] The swipe experience tells a visual STORY, not just a text story
