## The Core Lesson (March 17, 2026)

Video is NOT static image generation. You can't just prompt → generate → ship.

Video is a KITCHEN with stations. Each station needs mastery. They synergize to produce ONE excellent piece.

## The Production Flow
Brief → Script → NANO Reference → Prompt Engineering → Generation → Editorial Kitchen (Editor + Colorist + Typographer + Sound + VFX) → QC → Feedback Loop

2. **TYPOGRAPHY** — using basic FFmpeg drawtext. Need Remotion kinetic text with brand type system.
3. **SOUND DESIGN** — only flat BGM. Need music-first editing, ducking, SFX, silence beats.
4. **EDITORIAL RHYTHM** — mechanical cuts. Need beat-synced, musical pacing.
