## Project: Jade Oracle
Working dir: `/Users/yi-vonnehooi/Desktop/BRANDS/Jade Oracle/`
Brand DNA: `brand-guide/JADE-ORACLE-BRAND-DNA.md`

### Concept
AI oracle character "Jade" — luxury digital oracle house. The Byredo/Aesop of spiritual guidance. First luxury oracle brand. International readings. NOT "psychic" (too commercial) — "oracle" (truth revealer).

### Jade Character
- Late 20s, ethnically ambiguous, jade green eyes (trademark), silver hair streak
- Warm low alto voice (ElevenLabs), measured pace, slight non-specific accent
- Quiet luxury + ancient priestess aesthetic. Never costume-y.
- "She already knows the answer but wants you to discover it yourself"

### Palette
- Midnight Ink #0A0F1A + Antique Gold #C9A96E + Warm Ivory #F5F0E8
- Deep Emerald #1B4332 (Jade's color) + Soft Blush #E8D5C4 + Oxidized Bronze #8B6F47
- NO purple gradients, NO galaxy, NO neon

### Typography
- Display: High-contrast Didone serif (Claritle/Maglite/ET Emilia Grace)
- Body: Suisse Int'l or Neue Haas Grotesk
- No whimsical/hand-drawn scripts

### Logo
- Primary: "JADE" wordmark (Didone, J with subtle serpent/crescent flourish)
- Secondary: Abstract sigil/monogram (sacred geometry proportions)
- Lockup: Wordmark + "THE ORACLE"

### Content Strategy
- Reels for discovery, Carousels for conversion
- 7 content categories: PAC (pick-a-card), ZOD (zodiac), ORC (oracle wisdom), THD (talking head), TST (testimonial), RDG (reading sample), RIT (ritual)
- Grid rhythm: DARK | CARD | JADE (every 3rd post = Jade's face)

### Technical Pipeline
- Face lock: LoRA (0.6) + PuLID Flux II (0.8) + ControlNet
- Talking head: RunningHub InfiniteTalk
- Voice: ElevenLabs custom voice
- Static: NANO + PIL post-process + grain

- Free content → $5-15 Patreon → $100-150 readings → $300-500 deep sessions
- VIP $200/month unlimited → Custom oracle deck $45-65 → Course $200-500
- Psychic Samira benchmark: $660K+ lifetime from 11K customers
