# Bloom & Bare — Autonomous Social Media Director

**Started**: 2026-03-21
**Status**: Architecture phase

## The Vision
Fully autonomous Social Media Director that runs BB's Instagram without human input. The entire team in one system:
- Brand strategist (content calendar, festive awareness, Malaysian cultural context)
- Creative director (design production, photo curation, reference scraping)
- Copywriter (captions, hashtags, bilingual EN/CN)
- Designer (NANO production, post-processing, grid aesthetic)
- Social media manager (posting via IG Graph API, scheduling)
- Performance analyst (forensics, reporting, optimization loop)

**User input**: Only for ad-hoc special requests (events, promotions, announcements)
**Everything else**: Autonomous — never miss a festive greeting, never have a gap day

## Requirements
- 2 posts per day minimum
- Malaysian festive calendar awareness (Hari Raya, CNY, Deepavali, Christmas, school holidays, etc)
- Bukit Jalil, KL context
- Bilingual EN + CN
- Grid aesthetic (BOLD | PHOTO | CLEAN row rhythm)
- Performance forensics + action loop
- Brand-agnostic intelligence — any brand can use the system

## Content Mix (per week, 14 posts)
- Photo posts (real BB photos from library) — 4-5/week
- Illustrated/mascot posts (NANO generated) — 3-4/week
- Festive/seasonal greetings — as needed
- Educational/tips — 2/week
- Engagement (questions, polls, interactive) — 2/week
- Promotional (workshops, events, parties) — 1-2/week

## Photo Library
- UV glow play: 13 photos (added 2026-03-21)
- Need to catalog all existing BB photos

## Key Trigger
User frustrated that today (Hari Raya 2026-03-21) BB has no post. This system must prevent that from ever happening again.

## Technical Stack Needed
- Instagram Graph API (posting + insights)
- Facebook Page token for BB
- Content calendar engine (cron/scheduler)
- Malaysian festive calendar database
- Photo library management
- NANO design production (existing)
- Post-processing pipeline (existing)
- Copy generation engine
- Performance forensics engine
- Reporting + optimization loop
