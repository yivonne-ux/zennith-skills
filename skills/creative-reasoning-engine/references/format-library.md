## Reference Sourcing Rules

1. Every ref must be the ACTUAL FORMAT — real iPhone Notes, real calculator, real receipt
2. All refs must be 9:16 native (or 4:5 for feed posts)
3. Score every ref on 5 dimensions (Typography, Composition, Color, Hook, Relevance) — reject < 7 avg
4. Organize by format number: `references/formats/F01-iphone-notes/`, `F02-calculator/`, etc.
5. Refresh every 14 days (new app updates, seasonal trends)
6. Tag as "universal" or "brand-specific"
