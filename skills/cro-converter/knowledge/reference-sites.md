# Reference Sites — Forensic Analysis

## Co-Star (costarastrology.com)
- **Design**: Pure black + white. Zero color. Anti-astrology aesthetic.
- **Typography**: Modified neo-grotesque sans. All lowercase signature.
- **Tech**: React SPA, client-rendered
- **Trust**: Through aesthetics, not badges. NO testimonials, NO star ratings. NASA mentioned casually.
- **Conversion**: Single CTA (download app). No pricing on homepage. No pop-ups.
- **Bold choice**: "We're the anti-astrology astrology brand"
- **Steal**: Anti-template aesthetic, typography-as-brand, radical simplicity, cultural positioning

## Moyra (moyra.app)
- **Design**: Modern app landing page, warm palette
- **Tech**: Next.js SSR
- **Conversion**: App-first (iOS/Android download CTAs)
- **Steal**: Mobile-first thinking, clean section hierarchy

## Moon Omens (moonomens.com)
- **Design**: Alice serif + HK Grotesk sans. Pink accent #f5b0cf. WordPress + WooCommerce.
- **Products**: Books, crystals, memberships (MemberPress)
- **Content**: Articles by category (Astrology, Spirituality, Health, Numerology)
- **Community**: Thrive Comments, WP ULike, social links
- **Bold choice**: "We're the spiritual magazine"
- **Steal**: Two-font system, content-as-authority, product ecosystem, mega-menu navigation, editorial warmth

## Psychic Samira (my.psychicsamira.com)
- **Design**: Shopify. Cream bg rgb(252,251,244). HOT PINK CTAs rgb(251,7,151). Zero section spacing. Zero border radius.
- **Store**: Originally "yourpupstar" (pet brand pivot)
- **Scale**: 30+ country-specific domains. Massive international operation.
- **Trust stack**: Trustpilot + Loox photo reviews + Instagram feed (NfCube) + Gorgias chat + payment badges
- **Conversion stack**: Hot pink CTAs + zero spacing + SMS (Postscript) + referral (Social Snowball) + express checkout
- **Tracking**: Facebook Pixel + TikTok Pixel + Google Analytics + WeTracked + Axon (AppLovin)
- **Bold choice**: "We're the conversion machine"
- **Steal**: Multi-layered trust stack, photo reviews, hot-contrast CTAs, SMS capture, referral program, zero section spacing, full attribution tracking

Not brutalist (Co-Star). Not magazine (Moon Omens). Not conversion machine (Samira). A WORLD — where design IS the trust signal and every element feels like Jade chose it herself.
