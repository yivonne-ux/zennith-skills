# Grab Listing Optimizer — Complete Loop

## The 10-Step Flow

```
CLIENT DUMPS RAW MATERIALS
         ↓
1. ORGANIZE    — Audit files, map best-per-item, classify (food/drink/rice)
         ↓
2. PHOTOS      — Gemini AI catalog regeneration + QA gate + visual review loop
         ↓
3. COPY        — Bilingual names (EN+CN), emoji, appetite-triggering descriptions
         ↓
4. STRUCTURE   — 5-8 categories, best sellers first, logical flow
         ↓
5. PRICING     — Anchor high, value sweet spot, .90 rounding, S/L variants
         ↓
6. PWP/UPSELL  — Purchase combos, add-on prompts, free delivery threshold
         ↓
7. BRAND       — Store name, description, heritage story, USP tags
         ↓
8. PROMOS      — Always-on (new customer, free delivery), campaigns, flash sales
         ↓
9. REVIEWS     — Response templates (5/4/3-star), growth plan, cards in bags
         ↓
10. UPLOAD     — Batch upload via Grab Merchant portal + post-upload audit
         ↓
HIGH-CONVERSION GRAB STORE LIVE
```

## Output Package (per merchant)

```
~/Desktop/Grab Listing Output/{Merchant}/
├── photos-final/                     ← Catalog-grade 800x800 photos
├── reports/
│   ├── COMPLETE-LISTING-PACKAGE.md   ← THE master doc — everything in one place
│   ├── LISTING-COPY.md               ← Item-by-item copy to paste
│   └── upload_manifest.json          ← Machine-readable for automation
```

## COMPLETE-LISTING-PACKAGE.md Structure

1. **Store Identity** — Name, description, brand story, tags
2. **Menu + Pricing** — Every item with price + strategy reason
3. **PWP + Upselling** — Combos, add-on triggers, free delivery threshold
4. **Promotions** — Always-on + campaigns + flash sales
5. **User Reviews** — Response templates + growth plan
6. **Upload Checklist** — Step-by-step to go live

## Pricing Framework

| Tier | Price Range | Purpose |
|------|------------|---------|
| Premium Anchor | RM35-50+ | Makes mid-tier look affordable |
| Core Value | RM13-22 | Where 70% of orders land |
| Entry Point | RM9-13 | Captures budget/trial customers |
| Add-Ons | RM5-10 | Impulse, margin boosters |
| Drinks | RM3-10 | Low friction add-on to hit free delivery |

## PWP Framework

| Combo Type | Discount | Goal |
|-----------|----------|------|
| Main + Side | -RM3 | Increase AOV +RM8 |
| Upgrade | Show price gap only | 38% upgrade rate |
| Family Set | -RM8-12 | AOV RM50+ |
| Tea Break | -RM3 | Off-peak fill |

## Review Response Rules
- 5-star: Thank + mention specific dish + brand story hook
- 4-star: Thank + ask for feedback + promise improvement
- 3-star: Apologize + offer fix + DM request
- NEVER argue. NEVER be defensive. ALWAYS offer solution.
