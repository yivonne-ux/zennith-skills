## PROMPT ARCHITECTURE (7 layers)

```
[OUTPUT TYPE] + [QUALITY ANCHOR] + [SUBJECT] + [STYLE REFERENCE] +
[COLOR/MATERIAL] + [LIGHTING/MOOD] + [ANTI-PATTERNS]
```

### Layer Definitions:
1. **Output type**: "Professional brand identity presentation" / "Macro photograph" / "Editorial flat lay" / "Luxury emblem logo"
2. **Quality anchor**: "Pentagram-quality" / "Kinfolk editorial" / "Cocorrina level craft" / "Museum-quality"
3. **Subject**: The actual thing being depicted
4. **Style reference**: "Like Aesop's apothecary aesthetic" / "Bottega Veneta photography direction"
5. **Color/material**: Specific hex codes, material descriptions (600gsm cotton, matte black, gold foil)
6. **Lighting/mood**: "Warm directional side light, 3200K" / "Soft natural window light"
7. **Anti-patterns**: "No gradients, no 3D, no AI-generated feel, no stock photo energy"

## PROVEN PROMPTS BY CATEGORY

### LOGO / EMBLEM
```
Professional luxury emblem logo on [BACKGROUND]. A sophisticated [MARK TYPE]
for a brand called [NAME]. [DETAILED MARK DESCRIPTION — shapes, letters,
construction]. Rendered in [COLOR] on [SURFACE]. [QUALITY REFERENCE].
Below: [NAME] in [TYPE STYLE]. Professional brand identity presentation
showing the mark at [CONTEXTS]. [MOOD]. [ANTI-PATTERNS].
```

### BRAND COLLATERAL FLAT LAY
```
Professional brand identity [flat lay / system presentation] on [SURFACE].
[NUMBER] items arranged with editorial precision and controlled asymmetry:
1) [ITEM 1 with material detail], 2) [ITEM 2], 3) [ITEM 3].
[LIGHTING DIRECTION]. The [MATERIAL QUALITY] communicates [FEELING].
[PHOTOGRAPHY STYLE REFERENCE]. This is what [BRAND]'s [CONTEXT] looks like.
[MOOD SENTENCE].
```

### PRODUCT PHOTOGRAPHY (oracle cards, packaging)
```
Product photograph of [PRODUCT] on [SURFACE]. [DETAILED MATERIAL
DESCRIPTION — stock weight, finish, edge treatment]. [SURROUNDING PROPS
— max 3, each with material detail]. [LIGHTING — single source direction,
color temp]. Shot with [LENS] at [APERTURE]. [FILM QUALITY]. The
[MATERIAL QUALITY] IS the brand. [MOOD].
```

### LIFESTYLE / IG CONTENT
```
A photograph that looks like it was casually taken [CONTEXT]. [PERSON
DESCRIPTION — clothing, pose, NOT posing]. [ENVIRONMENT — 3-4 specific
details]. Natural [TIME OF DAY] light. The photo feels [AUTHENTIC
QUALITY — intimate, unposed, candid]. No filters, no heavy editing.
The color palette is naturally [PALETTE]. This is not a brand photo —
this is someone's beautiful [DAY/MOMENT]. [CAMERA STYLE].
```

### STORE / ENVIRONMENT
```
Architectural photograph of [SPACE TYPE]. [KEY ARCHITECTURAL ELEMENT].
[MATERIAL PALETTE — 3-4 materials]. [LIGHTING — ambient, natural, warm].
The interior [DESIGN PHILOSOPHY]. [WHAT THE SPACE COMMUNICATES].
[REFERENCE — "Aesop store meets..." ]. [TIME OF DAY]. Architectural
photography, [LENS].
```

### MACRO / DETAIL SHOT
```
Extreme close-up macro photograph of [OBJECT/DETAIL]. [MATERIAL AT
MICROSCOPIC LEVEL — texture, fiber, impression, reflection]. [LIGHTING —
warm directional from SPECIFIC DIRECTION creating shadows]. The [CRAFT
QUALITY — letterpress impression, foil edge, paper fiber, ink density].
Shot with [MACRO LENS] at [APERTURE]. Shallow depth of field. Film
grain. [WHAT THIS DETAIL COMMUNICATES — "the craft speaks"].
```

## ANTI-PATTERN PHRASES (add to every prompt)

- "No gradients, no 3D effects, no shadows"
- "Not a digital rendering — a physical artifact"
- "No stock photo energy"
- "No AI-generated feel"
- "Professional, not template"
- "Flat vector, no effects" (for logos)
- "Film grain, not digital clean" (for photography)
