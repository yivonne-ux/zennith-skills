## 7. Image Generation Workflow (Lovart + Nano Banana Pro)

### Process
1. Reference image → Lovart analyze
2. JSON prompt from Gemini → paste in Lovart
3. Ctrl+click specific parts + add comments
4. Iterate based on image outcome
