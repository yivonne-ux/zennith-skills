## SECTION 9: AD FATIGUE MANAGEMENT AT 30+ ADS/DAY

### The fatigue timeline
| Week | What happens |
|------|-------------|
| 1-2 | Peak performance |
| 3-4 | CPMs +29%, CTR -35% |
| 4+ | Severe degradation |

### Refresh cadence
- Cold prospecting: every 2-3 weeks
- Retargeting: every 4-6 weeks
- ASC: minimum 5 new assets per bi-weekly cycle
- Always have NEXT batch ready before current batch fatigues

### Modular creative system (the key to 30+/day)
Build interchangeable components:
1. **5-7 hook variations** (different opening frames/lines)
2. **3-4 body narratives** (different angles/stories)
3. **3-4 CTA endings** (different offers/urgency)
4. **Mix-and-match** = dozens of unique combinations

### The diversity rule
For every winner, create 3-5 genuinely different variations:
- Same message, different FORMAT (static → video → carousel)
- Same product, different ANGLE (close-up → lifestyle → UGC)
- Same offer, different EMOTION (fear → aspiration → social proof)

## SECTION 11: PRODUCTION PIPELINE FOR 30+ ADS/DAY

### Daily production workflow
```
Morning: Review yesterday's performance data (kill/scale decisions)
├── Kill: frequency >2.5, CTR -20%, CPA +30%
├── Scale: increase budget 20% on winners
└── Flag: creatives entering week 3 (fatigue incoming)

Midday: Produce new batch
├── 5 static ads (4:5, 1080×1350) — NANO pipeline
├── 3 carousel ads (1:1 cards) — mix of food + copy
├── 2 Reels/video (9:16, sound-on) — UGC or motion
└── = 10 genuinely distinct concepts per day

Afternoon: Launch in testing campaign (20-30% budget)
└── RM20-40 per concept, 3-5 day evaluation period

Weekly: Graduate winners to ASC scaling campaign
```

### NANO pipeline (static ads)
Proven from cn-ads-v4 (24 variants, 4 fix rounds):
1. Reference image + food photos → NANO Banana Pro Edit (2K resolution)
2. 7-layer prompt: SAFETY → BRAND → FOOD → TYPO → BADGE → SCENE → OUTPUT
3. Post-process: resize 1080×1350 → place_logo(auto) → add_grain(0.016)
4. Cost: ~$0.15/image = ~$1.50/day for 10 statics

### Anti-patterns (hard-learned from cn-ads-v4)
1. No AI humans in scenes — 100% obviously fake
2. No shrinking type to fit more food — massive type = expensive feel
3. No blanket rules — analyze per variant by funnel stage
4. No English text in NANO prompts — it leaks into visible design
5. No multi-food layouts — NANO reinterprets all food. Single food + strong type wins.
6. No hex codes or pixel values in prompts — NANO renders them as text

## SECTION 17: CREATIVE PRODUCTION FACTORY

### The 3-3-3 Method (Industry Standard)
3 concepts × 3 hooks × 3 formats = **27 ads per cycle**
- Cycle = weekly
- Test budget: RM150+ per concept (~RM450/week testing)
- **80-90% of concepts will fail.** 10-20% are winners. This is normal.

### Kill / Scale Criteria
| Timeframe | Action |
|-----------|--------|
| 24 hours | Kill if CTR < 0.5% |
| 3 days | Kill if CPA > 2x target after 10K impressions |
| 7 days | Final decision — promote winners, kill rest |
| Scale | Increase budget 20% every 48-72h using ORIGINAL Post ID |

### Creative Lifespan
- Average: **14-21 days** before fatigue
- After **4.2 exposures** per user, conversion drops 45%
- Frequency > 2.5 for prospecting = immediate refresh
- CPMs inflate 29% after 3-4 weeks without refresh
- **Refresh ≠ new concept.** Swap food photo, change headline color, flip layout = extends life

### Andromeda Diversity Rules
Simply swapping headlines on the same image is NOT enough. Must change at least ONE core dimension:
1. Message frame (weight loss vs time saving vs taste)
2. Authority source (nutritionist vs customer vs founder)
3. Proof type (number vs testimonial vs comparison)
4. Voice/POV (first person vs second person vs third)
5. Visual composition (flat-lay vs hero vs lifestyle vs text-dominant)
6. Format (static vs carousel vs video)
