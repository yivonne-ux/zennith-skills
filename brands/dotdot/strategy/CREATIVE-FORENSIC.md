**Aesthetic**: "German Precision Meets Hong Kong Warmth"
- Not clinical/cold (that alienates elderly)
- Not cute/childish (that undermines medical credibility)
- The sweet spot: **Clean, warm, trustworthy, accessible**

**Color Direction**:
```
Primary: DotDot Teal #4DBFB8 (60%) — trust, calm, health
Secondary: DotDot Orange #E8752A (25%) — warmth, energy, the product
Accent: Cream #FFF9F0 (10%) — approachability, softness
Text: Near-black #1A1A1A (body) + White (on dark)
Product-specific: Each product gets its own card color
```

**Typography Direction**:
- Display: Bold rounded sans-serif (approachable, not clinical)

## 5. CALIBRATION PLAN

### Phase 1: Style Lock (before any batch production)

1. **Scrape 20-30 references** from Tier 1+2 accounts via Chrome/instagrapi
2. **Classify each**: context / design / both
3. **Select 8-10 "Image 1" candidates** — approved references for NANO edit-first
4. **Generate 6 test outputs** (one per template type T1-T6)
5. **Human review**: yes/no on each, with specific feedback per dimension
6. **Iterate** until 5/6 pass on first review

### Phase 2: Color Calibration

1. Generate same design in 3 color variations (teal-dominant, orange-dominant, cream-dominant)
2. Lock the palette distribution that feels most "DotDot"
3. Test elderly readability — show to someone 55+ and ask "can you read this at arm's length?"

### Phase 3: Typography Calibration

1. Test 3 Chinese font weights for headlines (bold vs black vs heavy)
2. Test body text sizes (minimum readable for elderly)
3. Lock font pairing

### Phase 4: Mascot Direction

1. Option A: Evolve existing kawaii mascot → 3D Pixar render
2. Option B: New 3D character designed for the brand
3. Generate 3 test frames, human picks direction
4. Lock character sheet (front, side, expressions, poses)

### Success Criteria

The calibration is DONE when:
- Any random person shown 3 DotDot posts can identify them as the same brand
- An elderly HK person can read every post at arm's length
- The grid looks intentional and premium when squinted
- Exercise demo content maintains its proven high-engagement format
- Medical compliance gate passes on every output
