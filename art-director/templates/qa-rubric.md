# Art QA Rubric (quick)

Score each 1–5:

## 1) Style consistency
- Matches refs: palette, brush/texture, lighting, composition

## 2) Legibility (phone)
- Text readable at small size
- Enough negative space
- Contrast OK (cream/white text + subtle shadow if needed)

## 3) Brand fit
- Gaia/Pinxin mood + values consistent
- No off-brand colors or weird elements

## 4) Visual impact
- Scroll-stopping focal point
- Emotional tone correct

### If score <4, fix next
- Palette anchors: specify 5–8 colors + ban neon
- Texture: add "paper grain / gouache / watercolor"; ban "3D, glossy, plastic"
- Lighting: specify "soft rays / diffused"
- Composition: instruct "leave clean area for text"
