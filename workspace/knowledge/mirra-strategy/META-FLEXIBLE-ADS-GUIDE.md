# Meta Flexible Ads — Deep Research (March 2026)

## What Are They?
Single ad container: upload up to 10 images/videos → Meta auto-selects best FORMAT (single image, video, or carousel) per user per placement. Replaced Dynamic Creative for Sales/App Promotion objectives (June 2024).

## Performance
- **Conversion rate: 2.40% vs 1.24% standard** (nearly 2x)
- 25-40% performance lift over non-flexible
- Best stack: **Flexible Ads inside ASC** = maximum automation

## Key Differences
| Feature | Flexible Ads | Dynamic Creative | Advantage+ Creative |
|---------|-------------|-----------------|-------------------|
| Input | Multiple assets (up to 10) | Multiple assets mixed | Single asset, AI-modified |
| Format selection | Auto (image/video/carousel) | No format selection | No format selection |
| Objectives | Sales, App Promo ONLY | Traffic, Leads, Awareness | All objectives |
| CTA testing | No (1 CTA only) | Yes (multiple CTAs) | No |
| Carousel auto-gen | YES | No | No |

## Setup (Exact Steps)
1. Create → Sales objective
2. Ad level → Format → select "Flexible"
3. Upload 5-10 diverse images/videos
4. Create up to 3 creative groups
5. Add up to 5 primary text, 5 headlines, 5 descriptions per group
6. 1 URL per group, 1 CTA per ad
7. Enable Advantage+ Creative on top (optional)

## Asset Specs
- Images: min 1080×1080, both 1:1 and 9:16 recommended
- Videos: 15 sec max (qualifies all placements), MP4/MOV
- Same size/ratio across assets for best carousel auto-gen

## Critical Rules
1. NOT a testing tool — per-asset reporting is broken. Test concepts with separate ads.
2. Don't upload 10 similar images — Andromeda penalizes >60% similarity
3. Include video assets — video wins in Reels/Stories placements
4. Use creative groups to prevent incoherent combos
5. Refresh 20-30% of assets every 2-3 weeks
6. Cannot customize creative by placement — upload both 1:1 and 9:16

## How It Fits the 4-Campaign Structure
- **Test campaign:** Use separate standard ads (clean variable isolation)
- **Scale campaign:** Graduate winners INTO Flexible Ads (extends creative lifespan)
- **ASC campaign:** Flexible Ads are the PRIMARY format here
- **Retarget campaign:** Flexible Ads work well (diverse format = diverse touchpoints)
