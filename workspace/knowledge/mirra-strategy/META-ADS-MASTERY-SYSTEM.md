## 2. DECISION ENGINE

### 2.1 Campaign Objective Selection
```
DECISION TREE:
├── Want purchases/sales? → OUTCOME_SALES
│   ├── Have CAPI? → Optimize for PURCHASE event
│   └── No CAPI? → Optimize for CONVERSATIONS (Meta models conversions)
├── Want leads/conversations? → OUTCOME_ENGAGEMENT or OUTCOME_LEADS
│   ├── WhatsApp? → CONVERSATIONS optimization
│   └── Forms? → LEADS optimization
├── Want brand awareness? → OUTCOME_AWARENESS (rarely recommended)
└── Want traffic? → NEVER USE FOR CONVERSION-FOCUSED BRANDS
```

**CRITICAL LEARNING (Mar 2026)**:
- OUTCOME_ENGAGEMENT finds CHATTERS (people who reply)
- OUTCOME_SALES finds BUYERS (people who purchase)
- Same cost/conversation, DRAMATICALLY different conversion quality
- Mirra proved: 274 convos/day on ENGAGEMENT = 7 sales. Expected: SALES will convert 3-5x better.

### 2.2 Campaign Structure Decision
```
2026 BEST PRACTICE (Andromeda-optimized):
├── 1 CBO Campaign per objective per language
├── 1-2 Ad Sets (broad targeting, Advantage Audience ON)
├── 10-20 unique ads per ad set
├── Budget at campaign level
└── Let Andromeda find the audience via creative

WHEN TO USE ABO (ad set budgets):
├── Testing phase: control budget per creative batch
├── Retargeting: different budgets for hot/warm/cool
└── Language segmentation: EN vs CN need separate control
```

### 2.3 Bidding Strategy Selection
```
DECISION TREE:
├── Just starting / testing? → LOWEST COST (no cap)
│   └── Let Meta learn freely, gather data
├── Know your target CPA? → COST CAP
│   ├── Set at 1.2x your actual target CPA
│   ├── Best for scaling without losing ROAS
│   └── Meta adjusts bids dynamically around cap
├── Hard budget constraint? → BID CAP
│   ├── Strictest control, may limit delivery
│   └── Best for short campaigns with fixed margins
├── E-commerce with clear margins? → ROAS TARGET
│   ├── Set minimum ROAS threshold
│   └── Meta prioritizes high-value conversions
└── CTWA / WhatsApp? → LOWEST COST initially
    └── Switch to COST CAP once you know CPA
```

### 2.4 Audience Strategy
```
2026 REALITY:
- Interest targeting = DEAD (removed Jan 15, 2026)
- Exclusions = DEAD (removed Mar 31, 2026)
- Lookalikes = DEAD (replaced by ML models)

WHAT WORKS:
├── Broad targeting + Advantage Audience = DEFAULT
├── Custom audiences for retargeting only
├── First-party data (email lists, pixel, CAPI)
└── Creative IS your targeting (Andromeda reads creative to find audience)

AUDIENCE SIZE GUIDE:
├── Prospecting: 2-10M (Malaysia)
├── Retargeting: whatever your data gives you
└── CTWA: broader = better (Andromeda needs room)
```

## 3. KILL / SCALE CRITERIA

### 3.1 Ad Level
```
KILL (turn off ad) when:
├── Spent 2x CPA target with 0 conversions
├── CTR < 0.5% after 1,000+ impressions
├── Cost per result > 3x account average
├── Frequency > 3 on cold audience + CTR declining
└── Creative fatigue: frequency > 2.5 AND CTR dropped 30%+ from peak

SCALE (increase budget) when:
├── CPA < target for 3+ consecutive days
├── ROAS > target for 3+ consecutive days
├── CTR > 1.5% sustained
├── Frequency < 2 (room to grow)
└── Cost per result improving or stable over 7 days
```

### 3.2 Ad Set Level
```
KILL when:
├── Learning phase limited (stuck after 14 days)
├── All ads underperforming
├── CPA 2x+ above campaign average for 5+ days
└── Audience exhaustion: frequency > 4 on entire ad set

SCALE when:
├── Exited learning phase successfully
├── CPA at or below target for 7+ days
├── Multiple winning ads within the set
└── Scale 20% max every 48 hours (avoid learning reset)
```

### 3.3 Campaign Level
```
KILL when:
├── ROAS below 2x for 14+ days (for Mirra: below 4x)
├── All ad sets underperforming
├── Fundamentally wrong objective (e.g., ENGAGEMENT when need SALES)
└── Better campaign already proving superior

SCALE when:
├── ROAS > 4x for 7+ days
├── Multiple winning ad sets
├── Budget increase follows 20% rule
└── Maintain for 48hrs between each increase
```

## 4. CREATIVE INTELLIGENCE

### 4.1 Volume Requirements (2026)
| Daily Budget | Min Creatives | Refresh Cycle |
|-------------|--------------|---------------|
| < RM1,000/day | 10-15 | Every 2-3 weeks |
| RM1,000-3,000/day | 20-30 | Every 2 weeks |
| > RM3,000/day | 30-50+ | Weekly refresh |

### 4.2 Entity ID Awareness
- Visually similar ads (>60% similarity) get SAME Entity ID = treated as ONE ad
- Need <40% shared visual/audio features between ads
- 8-20 genuinely DIFFERENT concepts outperform A/B variations
- Different = different layout, different color, different angle, different format

### 4.3 Creative Testing Method (3:2:2)
```
3 Creatives × 2 Primary Texts × 2 Headlines = 12 combinations
- Test CONCEPTS, not audiences
- Use Flexible Ads (replaced DCO) for auto-testing
- Winner after 1,000+ impressions per combination
- Kill losers at 2x CPA with 0 conversions
```

### 4.4 Format Hierarchy (2026)
```
HIGHEST PERFORMING:
1. 9:16 Vertical Video (7% higher CTR in Feed)
2. UGC-style / lo-fi video
3. Static with strong hook (text-heavy, bold)
4. Carousel (CPA -30-50% for product showcases)
5. Collection ads (for catalog-based)

FOR FOOD/MEAL BRANDS:
1. Unboxing / first bite UGC
2. Before/after transformation
3. "What I eat in a day" format
4. Price comparison (vs grab/foodpanda)
5. Menu carousel (weekly rotation)
```

## 5. BUDGET & SCALING PLAYBOOK

### 5.1 Budget Formula
```
Minimum daily budget = Target CPA × 10 (for 10-conversion learning phase)
Example: CPA RM50 → RM500/day minimum per ad set

For CTWA:
Minimum daily budget = 10 × average CPC
Example: CPC RM5 → RM50/day minimum
```

### 5.2 Scaling Protocol
```
VERTICAL (increase budget on winners):
├── Max 20% increase per 48 hours
├── RM350 → RM420 → RM504 → RM605
├── Monitor CPA for 48hrs after each increase
├── If CPA spikes >30%, pause increase, wait 72hrs
└── Never change budget + creative at same time

HORIZONTAL (duplicate to new ad sets/campaigns):
├── Duplicate winning ad set with new budget
├── Slight targeting variation (age band, placement)
├── Keeps original untouched
└── Good for testing higher budgets without risking winner
```

### 5.3 Budget Gates (Mirra RM800K North Star)
```
CURRENT: ~RM1,500/day → ~RM45K/month
GATE 1: RM2,500/day → Requires ROAS 4x+ sustained 14 days
GATE 2: RM4,000/day → Requires CAPI implemented + ROAS 4x+
GATE 3: RM6,000/day → Requires 20+ winning creatives + ROAS 4x+
GATE 4: RM8,000/day → Requires diversified campaigns (EN+CN+retarget)
TARGET: RM5,500-6,500/day = RM165K-200K/month → at 4-5x ROAS = RM660K-1M/month
```

### 7.2 Proven Patterns (from data)
1. **OUTCOME_SALES > OUTCOME_ENGAGEMENT** for purchase intent (historical: RM2.18M revenue)
2. **CBO > ABO** for mature campaigns (old SUPER WIN was CBO)
3. **Broad targeting** works for KL/Selangor meal delivery
4. **Video KOL content** drives best cost/convo (ZiQian RM1.37, Sunny RM1.16)
5. **S19 Transformation** is #1 static (639 convos, RM0.87/convo)
6. **BoomBoom** is 12-month champion creative
7. **Post-Raya** is high-intent window for diet/health messaging
8. **Chinese-MY audience** less contested during Raya
9. **Multi-dish showcase + promo + urgency = purchase formula** (Pinxin: every ad with purchases shows 6-16 dishes + 买X送X + time limit)
10. **Static > gimmick video for PURCHASES** (Pinxin: RM310 on EGC videos = 0 purchases; statics = RM72-91 CPA)
11. **Brand-created urgency = festival urgency** (TNG Day RM72 CPA beat 元宵 RM91 CPA)
12. **MY market: 8-10 ads per RM400/day budget** (28 ads = noise; 5 = too narrow; 8-10 = sweet spot)
13. **MY audience fatigues faster** — creative refresh every 2 weeks (smaller pool than US, frequency climbs faster)
14. **Malaysian consumers respond to value/promo hooks** (买X送X) > aspirational/lifestyle messaging

### 7.3 Disproven Assumptions
1. ~~"More conversations = more sales"~~ → Quality > quantity. SALES objective gets better quality.
2. ~~"Raya kills performance"~~ → CPM actually LOWER, conversations UP. Issue was objective, not seasonality.
3. ~~"Need CAPI for SALES objective"~~ → Old campaigns worked fine without CAPI using Meta's conversion modeling.
4. ~~"EGC gimmick videos drive purchases"~~ → Entertainment ≠ conversion. 28 EGC shorts, RM310 spent on deployed ones, 0 purchases. Gimmick is the content, product is afterthought.
5. ~~"More ads = more learning"~~ → In MY market at RM400/day, 28 ads = RM14/ad = noise. Cut to 8-10 for proper signal.
6. ~~"US ad strategies apply directly to MY"~~ → MY has 75% cheaper CPM, smaller audience (2-5M vs 200M+), faster fatigue, stronger promo response. Adjust ad count, refresh cycle, and hook style.
7. ~~"Festival hooks are essential for urgency"~~ → Brand-created events (TNG Day) perform equal or better. Don't depend on calendar.

## 9. EXECUTION PROTOCOLS

### 9.1 New Campaign Launch Checklist
- [ ] Objective matches business goal (SALES for purchases)
- [ ] CBO with appropriate daily budget
- [ ] 1-2 ad sets with broad targeting + Advantage Audience
- [ ] 10-15 unique creatives (< 40% visual similarity)
- [ ] Correct geo targeting (delivery zone)
- [ ] Welcome message / chat template configured
- [ ] Attribution window set (7-day click + 1-day view)
- [ ] CAPI events configured (if available)
- [ ] Naming convention followed
- [ ] DO NOT TOUCH for 7 days after launch

### 9.2 Weekly Optimization Protocol
```
DAY 1-7: DO NOT TOUCH (learning phase)
DAY 7: First review
├── Check which ads have 0 conversions after 2x CPA spend → KILL
├── Check which ads have strong CPA → NOTE as winners
├── Check overall campaign CPA vs target
└── DO NOT change budget or targeting

DAY 14: Optimization window
├── Kill underperformers (2x CPA, 0 conversions)
├── If campaign exited learning: consider 20% budget increase
├── Plan next creative batch (2-3 week refresh cycle)
└── Compare SALES vs ENGAGEMENT performance

DAY 21+: Scaling decisions
├── Winners identified → scale 20% per 48hrs
├── Creative fatigue detected → prepare refresh batch
├── If ROAS > 4x sustained → open next budget gate
└── Document all learnings in Test Results Registry
```

### 9.3 Creative Refresh Protocol
```
WHEN: Every 2-3 weeks, or when frequency > 2.5 + CTR declining
HOW:
1. Identify winning CONCEPTS (not just winning ads)
2. Create variations of winning concepts (different format, angle, hook)
3. Add 3-5 new concepts for testing
4. Batch ALL new creatives into ad set in ONE session
5. DO NOT remove old winners unless clearly fatigued
6. Total active creatives should stay at 10-20 per ad set
```
