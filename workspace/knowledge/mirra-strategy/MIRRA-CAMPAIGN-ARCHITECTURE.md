## CAMPAIGN STRUCTURE (4 Campaigns)

### C1: MIRRA_TEST_Creative (ABO)
- **Objective:** Conversations (CTWA)
- **Budget:** 15% = RM195/day (Week 1) → RM315/day (Week 6)
- **Bid:** Lowest Cost (no cap)
- **Attribution:** 7-day click, 1-day view
- **Ad Sets:** 1 creative per ad set, RM40-65/day each. 5-7 ad sets live.
- **Kill:** CTR <0.5% at 24h, Cost/WA >RM12 after 7d, RM450 spend + 0 conversions
- **Promote:** Cost/WA <RM7 for 5 days, 15+ conversions → move to SCALE via Post ID

### C2: MIRRA_SCALE_Winners (CBO)
- **Objective:** Conversations (CTWA)
- **Budget:** 50% = RM650/day → RM1,050/day
- **Bid:** Cost Cap RM75-85/purchase
- **Ad Sets:** 1-2 broad ad sets, 5-8 proven winners each
- **Scaling:** +20% every 72h. Post ID preservation for social proof.

### C3: MIRRA_ASC_Advantage (Advantage+ Sales)
- **Launch:** Week 4 (not before)
- **Budget:** 25% = RM325/day → RM525/day
- **Existing customer cap:** 10-15%
- **Creatives:** 15-20 proven + new, all formats mixed

### C4: MIRRA_RT_Retarget (CBO)
- **Budget:** 10% = RM130/day → RM210/day
- **Tier 1 HOT:** WA started no order 1-7d (40%)
- **Tier 2 WARM:** Ad click/IG engage 8-30d (40%)
- **Tier 3 WIN-BACK:** Past purchasers 30-180d (20%)

## KILL / SCALE DECISION TREE

```
Ad runs 24h:
├── CTR < 0.5% → KILL immediately
├── CTR 0.5-1.5% → WATCH 3 more days
└── CTR > 1.5% → promising, continue

Ad runs 3 days (RM150-210 spent):
├── 0 conversions → KILL
├── Some conversions, CPA > 2x target → KILL
├── CPA unstable → WATCH 3 more days
└── CPA < RM80, ROAS > 3x → PROMOTE to SCALE

Ad in SCALE for 5+ days:
├── Cost/WA < RM7 sustained → SCALE +20% every 72h
├── Cost/WA RM7-10 → HOLD, monitor
├── Cost/WA > RM10 → check frequency
│   ├── Frequency > 2.5 → creative fatigue, rotate
│   └── Frequency < 2.5 → audience issue, test new geo
└── ROAS drops >30% after budget increase → REVERT immediately
```
