## Web Optimization Notes

### Priority Conversions (before Shopify upload)
1. **All PNGs should be exported as JPEG (quality 85-90)** for web performance. PNGs are 1.5-2.5MB; JPEG equivalents will be 150-400KB.
2. **Resize 2048px images to 1024-1200px** for web. Shopify recommended max is 2048x2048 but 1200px is sufficient for most placements.
3. **hero-main.png** is the highest priority -- resize to 1920x1072 and export as JPEG for fast hero load.

### Quick Conversion Command
```bash
# Convert all PNGs to optimized JPEGs (run from store-ready/)
for f in *.png; do
  sips -s format jpeg -s formatOptions 85 "$f" --out "${f%.png}_web.jpg"
done

# Resize large images to max 1200px width
for f in *_web.jpg; do
  sips --resampleWidth 1200 "$f"
done
```
