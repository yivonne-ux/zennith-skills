## WINNER A: "Your tarot reader can't do math"
**Format**: TikTok text overlay

### Anti-Pattern Check

| # | Anti-Pattern | Status | Notes |
|---|-------------|--------|-------|
| AP-15 | No hierarchy | PASS | Single line, one hierarchy level — correct for overlay |
| AP-16 | All caps everything | WATCH | Must NOT be rendered in all-caps — mixed case required |
| AP-28 | Text over busy images | WATCH | Thumbnail must have clear text overlay zone |
| AP-34 | Generic brand voice | PASS | This is distinctly Jade Oracle — no competitor could use this line |
| AP-46 | Sage-and-crystal starter pack | PASS | Zero wellness cliches |
| AP-47 | Unsubstantiated claims | WATCH | "Can't do math" is comparative — ensure it's clearly tongue-in-cheek, not literal |
| AP-48 | Appropriated sacred symbols | PASS | No sacred symbols referenced |
| AP-50 | Desktop-designed mobile content | PASS | 7 words — optimized for mobile |
| AP-52 | Text-heavy social cards | PASS | 7 words, well under 15-20 word max |
| AP-53 | Ignoring platform safe zones | WATCH | Text overlay must avoid top 15% and bottom 20% of frame |
| AP-59 | Designing for awards, not audiences | PASS | This is audience-first — designed to provoke engagement |

### Brand Voice Check (DNA.json)
- Tone "warm, wise, slightly mysterious": **PARTIAL** — this is more "cheeky" than "warm." Jade's voice is warm-casual with depth. This hook leans more provocative. Acceptable for TikTok where brand voice flexes toward edgier, but should NOT be used on IG/email.
- "Avoid" list: No cosmic imagery, no galaxy, no gothic — PASS
- Character personality (warm, empathetic, wise, approachable): **WATCH** — "can't do math" is slightly dismissive of other readers. On-brand for TikTok, off-brand for other channels.

### Quality Score
| Criterion | Score |
|-----------|-------|
| Stops the scroll | 10/10 |
| Platform-native | 9/10 |
| Brand differentiation | 10/10 |
| Brand voice alignment | 7/10 (slightly edgy for Jade) |
| Legal safety | 8/10 (comparative claim — use humor clearly) |
| **Overall** | **8.8/10** |

### Recommendations
1. Render in mixed case, NOT all-caps
2. Use on TikTok only — too edgy for Instagram or email
3. Video context should show Jade smiling/laughing to soften the edge
4. Consider adding "(she uses a 4,000-year-old algorithm)" as on-screen caption follow-up to substantiate

## WINNER B: "12 signs vs. 1,080 gates. You deserve the full picture."
**Format**: Instagram carousel first slide

### Anti-Pattern Check

| # | Anti-Pattern | Status | Notes |
|---|-------------|--------|-------|
| AP-09 | Font vomit | WATCH | Use max 2 typefaces — heading serif + body sans |
| AP-15 | No hierarchy | PASS | Two sentences create natural hierarchy — numbers bold, "you deserve" warm |
| AP-24 | Fear of white space | WATCH | First slide must breathe — numbers need space around them |
| AP-27 | Centered everything | WATCH | Left-align or center with intentional whitespace — don't just center-dump |
| AP-29 | No visual entry point | PASS | Numbers "12" and "1,080" are natural entry points |
| AP-34 | Generic brand voice | PASS | "You deserve the full picture" is distinctly empathetic Jade voice |
| AP-46 | Sage-and-crystal starter pack | PASS | No wellness cliches |
| AP-48 | Appropriated sacred symbols | PASS | "Gates" is direct QMDJ translation, used authentically |
| AP-50 | Desktop-designed mobile content | PASS | Short enough for mobile, natural line breaks |
| AP-51 | Inconsistent feed aesthetic | WATCH | Must match Jade Oracle jade/gold/cream palette |
| AP-52 | Text-heavy social cards | PASS | 13 words — well within limit |

### Brand Voice Check (DNA.json)
- Tone "warm, wise, slightly mysterious": **PASS** — "you deserve the full picture" is warm and empathetic. "Gates" introduces QMDJ terminology with mystery.
- Visual palette: Must use jade green (#00A86B), gold accent (#D4AF37), cream (#F5F0E8) — editorial clean layout
- Character personality: **PASS** — empathetic ("you deserve"), educational (numbers), wise (comparison)

### Quality Score
| Criterion | Score |
|-----------|-------|
| Stops the scroll | 9/10 |
| Platform-native | 9/10 |
| Brand differentiation | 10/10 |
| Brand voice alignment | 10/10 |
| Legal safety | 10/10 |
| **Overall** | **9.6/10** |

### Recommendations
1. Typography: Large serif numbers (12 / 1,080) with "vs." small between them
2. Second line ("You deserve the full picture.") in lighter weight
3. Background: Cream (#F5F0E8) with subtle jade accent, NOT busy imagery
4. Swipe indicator at bottom — "Swipe to see what QMDJ reveals"
5. Carousel should have 7-10 slides explaining QMDJ gate system

## WINNER C: "23,847 women started with a $1 question. Most came back because Jade knew things Google couldn't. Your turn."
**Format**: Meta ad primary text

### Anti-Pattern Check

| # | Anti-Pattern | Status | Notes |
|---|-------------|--------|-------|
| AP-34 | Generic brand voice | PASS | "Jade knew things Google couldn't" is ownable — no competitor can claim this |
| AP-47 | Unsubstantiated claims | WATCH | "23,847 women" — this number MUST be real or clearly aspirational. If fabricated, this is misleading advertising. |
| AP-52 | Text-heavy social cards | N/A | This is ad text, not visual — but keep it scannable |
| AP-58 | Dark patterns | PASS | No deceptive urgency, no false scarcity |
| AP-59 | Designing for awards | PASS | Direct-response focused |

### Brand Voice Check (DNA.json)
- Tone: **PASS** — "Jade knew things Google couldn't" has that "gently mysterious" quality. "Your turn" is warm and inviting.
- Competitor reference: Uses Jade's differentiator (real computation vs generic) without naming competitors — PASS
- Values alignment: "Ancient wisdom made accessible" — the $1 entry point communicates accessibility. PASS.

### Quality Score
| Criterion | Score |
|-----------|-------|
| Stops the scroll | 9/10 |
| Platform-native | 10/10 (perfect Meta ad structure) |
| Brand differentiation | 10/10 |
| Brand voice alignment | 9/10 |
| Legal safety | 7/10 (social proof number must be verified) |
| **Overall** | **9.0/10** |

### Recommendations
1. **CRITICAL**: The number 23,847 must be real or removed. Replace with actual customer count when available. If pre-launch, use "Thousands of women" or wait until real data exists.
2. Add "See more" expansion text below the fold: "Jade uses Qi Men Dun Jia (奇门遁甲) — the same 4,000-year-old strategic system Chinese emperors trusted with military decisions. It's not a vibe check. It's astronomical math mapped to your exact birth time. Get your $1 reading now."
3. Creative pairing: Lifestyle image of Jade holding tea, looking at camera with warm smile
4. Test a version with "Jade saw things" instead of "knew things" — may be warmer
