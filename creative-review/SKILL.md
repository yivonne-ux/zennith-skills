# Creative Review Skill

## Purpose
Multi-agent quality review for all creative output before it goes live.
Ensures UI/UX quality, CRO effectiveness, and brand DNA alignment.

## Review Checklist

### UI/UX Review (Iris)
- [ ] Visual hierarchy clear (what do you see first/second/third?)
- [ ] Typography hierarchy (3+ size levels, consistent)
- [ ] Color usage intentional (matches brand DNA)
- [ ] Whitespace balanced (not cramped, not empty)
- [ ] Grid alignment consistent
- [ ] Interactive elements obvious (buttons look clickable)
- [ ] Loading states present
- [ ] Error states handled
- [ ] Mobile responsive
- [ ] Animations enhance (not distract)
- [ ] Accessibility (contrast ratios, focus states)

### CRO Review (Hermes)  
- [ ] Clear value proposition above fold
- [ ] Primary CTA visible and compelling
- [ ] Social proof present (testimonials, logos, numbers)
- [ ] Trust signals (security, guarantees)
- [ ] Urgency/scarcity appropriate
- [ ] Form fields minimal
- [ ] Page load speed < 3s
- [ ] No friction in user flow
- [ ] Exit intent considered

### Brand DNA Review (Dreami)
- [ ] Matches Design DNA file
- [ ] Typography matches brand fonts
- [ ] Colors match brand palette
- [ ] Tone of voice consistent
- [ ] Photography style aligned
- [ ] No off-brand elements

### Award-Winning Website Criteria (Awwwards)
- [ ] Design: innovation, creativity, visual impact
- [ ] Usability: navigation, content hierarchy, UX
- [ ] Creativity: original concept, unique approach
- [ ] Content: quality, readability, engagement
- [ ] Mobile: responsive, touch-friendly, fast
- [ ] Developer: clean code, performance, accessibility

## How to Run a Review
1. Take screenshots of every page/section
2. Run each checklist
3. Score: 1-10 per category
4. List specific issues with screenshots
5. Prioritize: Critical → High → Medium → Low
6. Feed results back to build agent (Taoz)
7. Log review in creative-learnings.jsonl
