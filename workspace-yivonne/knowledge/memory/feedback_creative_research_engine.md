## The Problem — Reference Fatigue (2026-03-22)

Used refs from Type folders by FILENAME without actually SEEING them. Result: 13/20 mediocre, 4/20 rejected. Only grid layouts with real food photos were acceptable because the food carried the design, not the reference.

## RELAPSE — Same Session (2026-03-22)

Even after documenting the problem above, RELAPSED in the same session:
- "Redesigned" D01/D06/D10 by swapping UI formats but using SAME curated folder refs
- D04 and D06 share the EXACT SAME reference file (search-bar-the-search-the-solution.jpg)
- Zero fresh scraping. Zero Pinterest/Behance research. Zero competitive check.
- User called it: "i dont feel the variation nor the wow factor anymore"
- Outputs were "competent but predictable" — no scroll-stopper energy
- Just rearranging existing refs ≠ creative research

**The real fix is NOT swapping formats. It's finding references that make you STOP scrolling.**
If the ref doesn't excite ME, the output won't excite HER.

Signs of reference fatigue:
- Same visual language repeating across variants (Notes app x5, same two-tone wall x3)
- Refs don't match the concept's INTENT (illustration ref → illustration output, dark ref → dark output)
- No design tension or surprise in outputs — everything feels "template-y"
- AI follows ref structure literally but misses the ENERGY the concept needs
- Output looks like "any meal delivery brand" not "Mirra"

**Why:** References are 50% of output quality. Bad ref = bad output, no matter how good the prompt is. I treated ref selection as a file-picking task instead of a creative direction task.

## The Fix — Creative Research Engine

### 1. ALWAYS visually audit refs before use
Before assigning ANY reference to a variant:
- OPEN the reference image (use Read tool to view it)
- Evaluate: composition, typography quality, color palette, design tension, visual hierarchy
- Score: would this win an award? If no, find a better one
- Check: does this match the CONCEPT's energy (not just its format)?
- Reject: if the ref is mediocre, the output will be mediocre

### 2. Detect reference fatigue — trigger scraping
Scrape new references when ANY of these are true:
- Library hasn't been refreshed in 14+ days
- More than 3 variants need the same format (e.g. 5 Notes ads but only 4 Notes refs)
- Output quality drops below 70% acceptance rate
- User flags quality drop (IMMEDIATE trigger)
- Starting a new brand or campaign direction
- Existing refs feel "seen before" or produce template-y outputs

### 3. Proactive creative research — ALWAYS be learning
Before every production batch, spend time on:
- **Pinterest**: search "[concept] ad design 2026", "[format] creative", "food ad editorial"
- **Behance/Dribbble**: latest typography, layout innovation, color trends
- **Meta Ad Library**: what are top-spending competitors running RIGHT NOW?
- **YouTube/Perplexity**: AI automation breakthroughs, new model capabilities, workflow innovations
- **Typography sites**: latest typeface releases, lettering trends, hierarchy innovations
- **Award sites**: D&AD, Cannes Lions, One Show — what's winning?

### 4. Reference scraping workflow
```
TRIGGER → RESEARCH → SCRAPE → CURATE → STORE

1. TRIGGER: fatigue detected or new campaign starting
2. RESEARCH: 30min browsing Pinterest/Behance/Meta Ad Library
   - Search terms specific to the concepts needed
   - Look for designs that make you STOP scrolling
   - Find 3-5x more than needed (curate down later)
3. SCRAPE: download high-quality refs (min 1080px wide)
4. CURATE: view EVERY image, score on 5 dimensions:
   - Typography (0-10): is the type design world-class?
   - Composition (0-10): does the layout have tension and hierarchy?
   - Color (0-10): does the palette evoke the right emotion?
   - Hook (0-10): would this stop someone scrolling?
   - Relevance (0-10): does this match our concept's energy?
   Score < 7 average = REJECT
5. STORE: organize by concept type, tag with scores
```

### 5. Cross-brand intelligence sharing
Every creative research session benefits ALL brands:
- A great typography ref found for Mirra might work for Pinxin
- A layout innovation from Bloom & Bare might work for Jade Oracle
- Store universal design intelligence in `creative-intelligence/` not brand folders
- Tag refs as "universal" or "brand-specific"

### 6. Quality gate — pre-generation audit
Before running ANY production script:
- [ ] Have I SEEN every reference image I'm about to use?
- [ ] Does each ref match the concept's ENERGY (not just format)?
- [ ] Is my reference library fresh (updated in last 14 days)?
- [ ] Do I have enough VARIETY in refs (no 2 variants sharing a ref)?
- [ ] Would a world-class creative director approve these refs?
If any answer is NO → stop and fix before generating.

**How to apply:**
- BEFORE every production batch, run the pre-generation audit checklist
- If refs are stale, initiate scraping BEFORE generating (not after rejection)
- Use Perplexity/WebSearch for trend research, Pinterest for visual refs
- Save research findings to creative-intelligence memory files
- This applies to ALL brands, ALL formats, ALL campaigns
