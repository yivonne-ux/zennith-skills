## FORENSIC BEFORE GENERATING — UNIVERSAL RULE

### The Mistake (PX March 27, 2026)
Generated 22 social posts for Pinxin using Pinterest illustration refs without:
1. Pulling PX's existing 1,062 posts to see what visual style they already use
2. Checking which visual styles get the most engagement for PX specifically
3. Looking at PX's existing vector character family (12 characters, semi-realistic style)
4. Using PX's OWN high-performing posts as reference images

Result: Illustrations in a Pinterest-derived style that doesn't match PX's existing brand visual language. First post (Sambal Petai static) got 5 likes = bottom-tier performance.

### The Rule
Before generating ANY visual content for ANY brand:

**Step 1: Performance Forensic**
- Pull last 50-100 posts via Graph API
- Analyze: engagement by format (video/carousel/static), by day, by hour
- Identify top 10 and bottom 10 posts
- Find: what FORMAT works? What CONTENT works? What TIME works?

**Step 2: Art Style Forensic**
- View the brand's existing posts visually — what's their current look?
- Check for existing brand assets: vector characters, illustration styles, photo treatments
- Identify which visual styles correlate with HIGH engagement
- Use the brand's own high-performing posts as REFERENCE IMAGES for new content

**Step 3: Gap Analysis**
- Compare existing content vs strategy document
- Identify what's MISSING (not what to replicate)
- New content should fill gaps while staying visually consistent with what works

### Reference Hierarchy (updated)
1. **Best**: Brand's OWN high-performing posts as reference
2. **Good**: Brand's existing visual assets (character family, templates, style guide)
3. **Acceptable**: Curated external references that match brand's established style
4. **NEVER**: External refs that contradict brand's existing visual language

### Compound Learning
PX data showed VIDEO = 134.9 avg engagement vs IMAGE = 14.3. We produced 100% static images. The forensic would have caught this immediately and shifted the production plan toward video-first.

**Why:** "Reference-first" doesn't just mean "use a reference image." It means "start from what ALREADY WORKS for this brand." The brand's own data is the ultimate reference.

**How to apply:** Run forensic BEFORE the content planning step. Let data inform the format mix, posting schedule, and visual style — not assumptions or strategy docs alone.
