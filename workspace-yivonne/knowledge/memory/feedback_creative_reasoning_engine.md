## Creative Reasoning Engine (2026-03-16)

### The Core Insight
You don't start from "what should we make." You start from "what does this persona already interact with daily?" Then hijack that familiar format and inject brand benefits.

### The 5-Step Chain
1. **Persona Behavioral Audit** — What apps, objects, content, emotions does she interact with daily?
2. **Format Library Cross-Reference** — Which familiar formats (UI mimicry, physical objects, cultural, editorial, data) match her behaviors?
3. **Concept Collision** — [Persona behavior] + [Familiar format] + [Brand benefit reframed] = Concept
4. **Hook Psychology Check** — Must trigger 1+ of: pattern interrupt, curiosity gap, identity confirmation, FOMO, value shock, recognition, voyeurism
5. **Diversity Check** — Must differ in 4+ of 7 dimensions from existing concepts (Andromeda suppression rule)

### Proven Concept Collisions (Mirra)
- Horoscope card: CN persona reads horoscopes → astrology card format → lunch benefits as fortune categories
- Tinder swipe: Young demo uses dating apps → swipe UI → lunch as a match (Grab = left, Mirra = right)
- Receipt haul: Persona does Shopee hauls → receipt format → weekly Mirra order as shopping haul
- Calculator: She mentally calculates spending → calculator screen → RM352/month savings proof
- Search bar: She Googles "healthy lunch KL" → search results format → Mirra as the answer

### The Formula
Hooks are DISCOVERED from data (forensic analysis of winners), not invented from brainstorming.
Formats are HIJACKED from persona's daily life, not chosen randomly.
Share triggers are ENGINEERED deliberately, not hoped for.
