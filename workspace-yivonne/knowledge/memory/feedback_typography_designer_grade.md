## Typography feedback (2026-03-12)

User: "the font is still very default looking, not designer adjusted font, like tracking, leading all into consideration, it looks sophisticated and expensive"

### Rules
1. **Serif headlines**: Must describe as "editorial luxury serif with high stroke contrast, ink-trap details, sharp wedge serifs, optical kerning at -20 to -40 thousandths, baseline optically adjusted, x-height ratio 0.48-0.52" — NOT just "elegant serif"
2. **Sans-serif body**: Must describe as "geometric humanist with 98% x-height ratio, open apertures, optically even letterspacing +15/1000em, ink-weight calibrated for screen" — NOT just "clean sans"
3. **Handwritten/script**: Must reference Pinterest-aesthetic brands: Spiritual Gangster (flowing calligraphic with intentional ligatures, thick-thin modulation), Nastygal (bold fashion script, attitude). NOT casual handwriting, NOT Comic Sans territory
4. **Tracking/leading in prompts**: Always specify exact values — tracking: -20/1000em for headlines, +15/1000em for body. Leading: 1.05-1.10 for headlines, 1.35-1.45 for body. These numbers MATTER.
5. **Overall**: Typography should look like it was set by a type designer who charges $200/hr, not auto-generated by Canva
