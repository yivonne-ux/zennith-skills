## CONFIRMED LOSERS — NEVER REPRODUCE
- **Data visualizations**: Weight tracker graph (RM83/c), calorie receipt (RM16/c)
- **Abstract motivational**: Sticky notes (RM20.9/c), generic quotes
- **Drop A Size format**: 0 conversations
- **Wedding niche**: Too narrow
- **Over-produced KOL v2**: Loses first-take energy
- **Generic menu showcase**: No hook, no transformation
- **Promo slideshows**: Too salesy without persona
- **Food comparisons** (vs grab/mamak): Doesn't convert
