### Critical Learnings
1. **Reference-first method** (ohneis): Generate image → decompose → Sora. PROVEN better output.
2. **Script before visuals**: Brief → Script → Reference → Generate. Not the other way around.
3. **Kitchen team**: Each station needs mastery. They work together on ONE dish.
4. **Steal like an artist**: Copy proven structures, adapt content. Don't invent.
5. **Sora ≠ food**: Use real library footage for food. Sora for atmosphere/lifestyle/impossible.
6. **Typography = critical gap**: FFmpeg drawtext is functional not designed. Need Remotion mastery.
7. **Content not ads**: The goal is engagement content that IS the brand, not product ads.
8. **Brand constant, format variable**: Every video different, every video unmistakably Mirra.
