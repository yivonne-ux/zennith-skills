## W3 PRODUCTION COMPOUND LEARNINGS (24 March 2026)

### ✅ YES — What Worked

1. **Two-pass workflow (4:5 → 9:16)** — Generates at 1080×1350 first (content in safe zone), then NANO extends edges. ALL content stays in Meta's visible area. PROVEN.

2. **Single hero dish = best food preservation** — When NANO gets 1 food cutout, it preserves it faithfully. RT-01 regen (sambal only) = perfect green petai beans. WA-02 regen (green curry only) = perfect purple eggplant.

3. **Explicit signature markers in prompts** — Writing "MUST show GREEN PETAI BEANS" or "MUST show PURPLE EGGPLANT" forces NANO to preserve the distinctive features. Without this, NANO remixes food into generic dishes.

4. **Proven winners as structural references** — Using BOFU-18 (boarding pass), BOFU-02 (receipt), BOFU-10 (lofi) as Image 1 references produced outputs that match the winning format structure.

5. **Identity gimmick formats work** — Bank statement, calculator, notification stack, delivery tracker, boarding pass, receipt, notes app — all produced clean, recognizable UI formats.

6. **Real pricing from website** — RM21.90/dish individual, combo pricing (RM129.50/5, RM185.20/8, RM350.40/16). Using real math in ads builds trust.

7. **Bold typography ads** — BOFU-24 (RM12++ 一盒) and RT-05 (还在想？别想了 下单吧) = strongest visual impact. NANO handles massive text well.

8. **Reference fatigue check before production** — Caught 8 overused refs before running. Swapped all to fresh references. Must do this EVERY batch.

### ❌ NO — What Failed

1. **NEVER generate at 9:16 directly** — Content sprawls into Meta overlay zones (top 14%, bottom 20%). Always 4:5 first.

2. **NEVER give NANO 3+ food cutouts** — With 3+ foods, NANO remixes them into generic Asian dishes. Loses signature markers. Max 2 food cutouts, ideally 1 hero.

3. **NEVER let NANO generate packaging** — WA-03 Round 1 showed AI-invented vacuum bag. Round 2 showed AI-interpreted packaging label. Real packaging exists at `pkg-*.png` in local-cache. ALWAYS use real photos.

4. **NEVER let NANO generate logos** — RT-04 showed "PINXIN" text generated by NANO at bottom-right (not the real logo). Logo MUST be composited in post-production via PIL only.

5. **NEVER write fictional per-dish prices** — BOFU-20 Round 1 had RM13.90-15.90/dish (doesn't exist). Real price: RM21.90/dish. Only use combo pricing in ads.

6. **NEVER trust NANO with text labels matching food** — RT-04 Round 1 labeled green curry as "肉骨茶". NANO matches labels to positions, not to actual food content. Must explicitly map: "Image 2=rendang, label=仁当猴头菇".

7. **NEVER audit food as "looks like food"** — Must pixel-match against real cutout signature markers (petai beans, purple eggplant, okra, black fungus, etc). "Looks like curry" is NOT sufficient.

8. **NEVER use the word "logo" in prompts** — BOFU-26 Round 1 rendered the literal word "logo" in the output. Avoid any meta-instructions that could leak.

### 🔧 PRODUCTION RULES (Enforced)

| Rule | Detail |
|------|--------|
| Workflow | 4:5 → blur-extend → NANO extend 9:16 → logo → grain |
| Food per ad | Max 2 cutouts. 1 hero preferred. |
| Food prompt | Include signature markers: "GREEN PETAI BEANS", "PURPLE EGGPLANT", etc |
| Packaging | REAL photos only from `pkg-*.png`. NEVER AI-generated. |
| Logo | PIL composite only. NEVER in NANO prompt. |
| Pricing | Combo pricing only (买8送8, RM12++/盒). NEVER per-dish RM. |
| Text labels | Explicitly map Image N = dish name in prompt. |
| Pre-production | Reference fatigue check. Asset existence check. |
| Post-production | Pixel-match food audit against signature markers. |
| Regeneration | Max 3 rounds. If still failing after 3 → escalate to human. |
