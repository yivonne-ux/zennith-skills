## PX FESTIVAL / CALENDAR BRAND FIT

**Why:** PX built 6 Hari Raya posts (prep, countdown, bundle, quotes). User caught it — "Pinxin suitable for Raya?" Answer: NO for a campaign, YES for 1 greeting.

### CORE FESTIVALS (go deep — multi-post campaigns)
- **CNY / 农历新年** — biggest. Multi-week campaign. Reunion dinner. Poon Choi spread.
- **Qingming / 清明** — family, ancestors, food as remembrance. Illustration-friendly.
- **Mid-Autumn / 中秋节** — mooncakes, family, lanterns. PX doesn't sell mooncakes but cultural post OK.
- **Nine Emperor Gods / 九皇爷** — vegetarian festival. PERFECT for PX. Go all-in.
- **Dumpling Festival / 端午节** — tradition, family cooking. PX has no dumplings but cultural post OK.
- **Chap Goh Mei / 元宵节** — end of CNY. Celebration, togetherness.
- **Dong Zhi / 冬至** — tang yuan, family reunion, warmth. Cultural illustration post.

### CROSS-CULTURAL (1 respectful greeting max, not a campaign)
- **Hari Raya Aidilfitri** — 1 greeting. PX has rendang (Raya food) but core audience is Chinese.
- **Deepavali** — 1 greeting. Light/festival of lights. Not PX's audience.
- **Christmas** — 1 greeting if relevant. Western holiday.
- **Merdeka / Malaysia Day** — 1 patriotic post. Multi-ethnic unity angle.

### COMMERCE EVENTS (evaluate fit)
- **Double-day sales (1.1, 2.2, 3.3, etc.)** — 4.4 Sale NOT confirmed relevant for MY market. Don't assume.
- **Black Friday / Cyber Monday** — Low relevance for MY Chinese food brand.
- **Ramadan** — PX is vegan but NOT halal-certified. Do NOT build Ramadan content.
- **Back to school** — Weak tie-in. "Busy mum" angle possible but thin.

### RULE
Before building any festive campaign: Ask "Is this PX's festival or PX's audience's festival?"
- If YES → go deep (3-5 posts, themed content, product tie-in)
- If cross-cultural → 1 respectful greeting, move on
- If NO connection → skip entirely
