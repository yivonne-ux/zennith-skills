# Reference Scraping Criteria — God Mode Framework

Universal. Not brand-specific. This is how to build a world-class reference library.

**Why:** Bad refs = bad outputs. The reference library IS the taste level. Garbage in, garbage out. A curated 200-image library beats an unfiltered 2000-image dump.

**How to apply:** Use the 7 filters when scraping new refs. Use the audit checklist when filtering existing libraries. Reject anything scoring <3/7.

---

## THE 7 FILTERS (pass ANY 3 = worth keeping)

### 1. CONVERSION ARCHITECTURE
The ad is BUILT TO SELL, not just pretty.
- Single clear CTA or implicit action driver
- Pain → solution flow visible in 2 seconds
- Social proof integrated (testimonials, numbers, before/after)
- Scarcity/urgency without being desperate
- Signal: ads running 90+ days in Meta Ad Library = converting

### 2. EDITORIAL PHOTOGRAPHY
Image quality passes in a magazine spread.
- Directional lighting with real shadows (not flat studio)
- Shallow depth of field / intentional focus
- 30-45° camera angles (not dead overhead, not straight-on)
- Two-tone depth (wall + surface, or gradient + texture)
- Signal: Bon Appétit, Kinfolk, Cereal Magazine energy

### 3. TYPOGRAPHIC CRAFT
Text treatment is DESIGNED, not placed.
- Mixed weights creating visual rhythm
- Intentional tracking/kerning (not default)
- High-contrast serif OR distinctive display type
- Text as a design element, not just information
- Signal: type you'd screenshot even without reading the words

### 4. LAYOUT INTELLIGENCE
Spatial composition teaches you something.
- Clear visual hierarchy (eye path in <1 second)
- Intentional negative space (DESIGNED empty, not just empty)
- Grid discipline — elements feel anchored, not floating
- Information density matched to funnel stage
- Signal: layouts you can reverse-engineer into reusable templates

### 5. BRAND COHESION
The ad FEELS like a brand, not a one-off.
- Consistent color story (2-4 colors max, palette feels intentional)
- Logo/brand mark integrated naturally (not stamped on)
- Voice/copy with personality (not generic marketing speak)
- Identifiable with logo removed
- Signal: D2C brands with 50+ ads in Meta Library (they've iterated)

### 6. EMOTIONAL TRIGGER
Creates a FEELING before the brain processes the message.
- Aspiration ("I want that life")
- Recognition ("that's literally me")
- Curiosity ("wait, what is this?")
- FOMO ("everyone's doing this and I'm not")
- Relief ("finally, someone gets it")
- Signal: ads you'd send to a friend. Ads that stop your scroll.

### 7. PRODUCTION VALUE
Craft level signals premium.
- Color grading that feels cinematic (not Instagram filter)
- Texture/grain that feels analog (not digital noise)
- Props that feel curated (not random)
- Surfaces that feel REAL (marble, linen, wood, concrete — not solid color fills)
- Signal: zoom in and the details hold up

## SCRAPING SOURCES (ranked by signal quality)

| Source | Why | What to grab |
|--------|-----|-------------|
| Meta Ad Library (90+ day runners) | Long-running = converting. Money talks. | Food, wellness, lifestyle, D2C |
| Foreplay.co / Swipe-Worthy | Curated high-performers with data | Top-performing static ads |
| Pinterest ("ad design" / "social media ad") | Design-forward, filtered by saves | High-save pins |
| Behance / Dribbble ("social media design") | Designer portfolios = editorial craft | Campaign mockups, brand systems |
| Instagram Explore (10K+ engagement brands) | Real-world proven | Carousel covers, single-image |
| AdSpy / BigSpy (paid) | Engagement + spend data | Filter by vertical + spend |

## INSTANT REJECT (disqualify immediately)

- Stock photography energy (Getty feel, generic people, white-bg cutouts)
- Canva templates (rounded corners everywhere, default gradients, template badges)
- Cluttered layouts (>5 text elements competing)
- Text-wall ads (paragraph copy as creative)
- Undesigned app/website screenshots (functional UI, not designed)
- Low resolution (<800px shortest side)
- Landscape ratio (need 4:5 or 9:16)
- Watermarked stock (Shutterstock, iStock marks)
- Icons/illustrations pretending to be ads (clip art energy)
- Price/offer as ONLY hook (no design merit)

## AUDIT SCORING

Score each ref 1-7 (how many filters does it pass?):

| Score | Action |
|-------|--------|
| 5-7 | **GOLD** — study deeply, reverse-engineer every element |
| 3-4 | **KEEP** — useful as layout/structure references |
| 1-2 | **REJECT** — delete, clutters the library |
| 0 | **Shouldn't be here** — instant delete |

---

## FOLDER ORGANIZATION

Organize refs by TYPE (what layout pattern they teach), not by brand or source:
- Type_01: Hero product shot (single product, editorial lighting)
- Type_02: Lifestyle/scene (product in context)
- Type_03: Split comparison (before/after, this vs that)
- Type_04: Testimonial/social proof
- Type_05: Process/steps (how it works)
- Type_06: UI/app screenshot (designed)
- Type_07: Typography-led (text IS the design)
- Type_08: Grid/mosaic (multiple products/options)
- Type_09: Offer/CTA (promotional with design merit)
- Type_10+: Emerging patterns as library grows
