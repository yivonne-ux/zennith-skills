Mirra is a PLANT-BASED brand. There is NO chicken, no beef, no meat.

- "Katsu curry" = plant-based katsu, not chicken
- "Bulgogi" = plant-based bulgogi, not beef
- "Nasi lemak" = plant-based version
- All dishes are plant-based alternatives

**Why:** User called out that ad copy said "real chicken katsu" — completely wrong. This destroys brand credibility. A plant-based brand cannot run ads claiming meat ingredients.

**How to apply:**
1. NEVER write "chicken", "beef", "pork", "meat" in any Mirra copy
2. Always verify dish names against actual Mirra menu
3. When describing food, focus on the dish name + calories + price, NOT ingredients you haven't verified
4. Match copy to the ACTUAL IMAGE, not just the filename
5. When in doubt, use generic terms: "bento", "meal", "dish" — not specific proteins
