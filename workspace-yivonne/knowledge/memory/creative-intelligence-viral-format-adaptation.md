## VIRAL FORMAT ADAPTATION ‚Äî UNIVERSAL METHOD

### THE FORMULA
**Copy FORMAT from proven viral. Change ART STYLE to brand. Change CHARACTER to brand. Keep CONTENT CONTEXT viral.**

### 3 INPUTS REQUIRED
1. **Format references** ‚Äî scraped from actual viral accounts (not imagined from descriptions)
2. **Brand style reference** ‚Äî the RICHEST, most polished artwork the brand has (not simplified assets)
3. **Character lock** ‚Äî exact visual spec confirmed by human (hair, clothing, accessories, age, expression)

### FORMAT CLASSIFICATION (do this BEFORE generating)
Map all scraped refs into layout types. Typical categories:
- **Dual-panel**: 2 poses stacked, text between (highest engagement on XHS)
- **Single portrait**: Close-up character + quote at bottom
- **Scene + overlay**: Character in environment + large text overlay
- **Action/prop**: Character holding item, doing activity
- **Contrast/comparison**: Before/after, cause/effect split
- **Dialogue**: Character talking to another person

### CONTENT CONTEXT ‚âÝ BRAND CONTEXT
- Viral granny posts are about LIFE WISDOM, not products
- The brand connection is the CHARACTER, not the content topic
- Posts should be 80% relatable life content, 20% brand-adjacent
- Copy the EMOTIONAL REGISTER of viral posts (sassy, healing, funny, sharp)
- NOT just warm/nice ‚Äî viral needs EDGE

### ART STYLE MATCHING ‚Äî THE #1 FAILURE POINT
- Using simplified vector character sheets ‚Üí flat clipart output
- Using brand concept art / hero artwork ‚Üí rich, dimensional output
- The quality of Image 2 (style ref) directly determines output quality
- Always use the brand's BEST illustration, not their simplest asset
- If output looks like "Canva template," the style ref is too weak

### QUALITY GATE
Art style mismatch = entire batch FAILS. Not "7/10 is OK."
Character detail wrong (hair, age, accessories) = entire batch FAILS.
Same layout repeated across all posts = FAILS (even if each post looks good individually).
Content too safe/generic = won't go viral (defeats the purpose).

**How to apply:** Before any viral format production for ANY brand, run through these checks. The method is brand-agnostic ‚Äî only the inputs change.
