## VIRAL FORMAT REFERENCE-FIRST — UNIVERSAL RULE

### The Principle
"Copy the FORMAT + COMPOSITION of proven viral content. Change the art style to [brand]."

### Why
- Imagined concepts = untested, unknown engagement potential
- Viral formats = PROVEN to get shares/saves/tags
- The joke structure, panel timing, emotional arc are already optimized by millions of viewers
- Only the visual style needs to change → brand's own character style

### How to Apply

**Step 1: Research viral formats in the brand's content niche**
- Search Pinterest, IG, TikTok, XHS for trending meme/comic formats
- Focus on formats with: high shares, tag-your-friend energy, save-worthy
- Document: panel layout, joke structure, emotional arc, what triggers sharing

**Step 2: Download visual references**
- Get the actual viral image as a composition template
- This becomes Image 1 in NANO (format/layout reference)

**Step 3: Generate with brand style**
- Image 1: Viral format reference (composition template)
- Image 2: Brand character assets (style anchor)
- Prompt: "Match the LAYOUT and PANEL STRUCTURE of Image 1. Match the CHARACTER STYLE of Image 2."
- NANO copies the proven format but renders in the brand's visual language
