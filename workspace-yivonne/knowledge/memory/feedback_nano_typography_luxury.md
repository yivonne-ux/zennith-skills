# NANO Typography — Luxury Body Copy Technique

**Problem**: NANO renders body copy in generic sans-serif fonts that look obviously AI. User finds this the weakest point.

**Solution**: ALL CAPS + VERY WIDE LETTER-SPACING is the single most effective trick.

## The Prompt Pattern

Body copy / info text prompt must include:
```
"— ALL CAPS with VERY WIDE letter-spacing (like Aesop packaging or Apple product page).
Premium geometric sans-serif, medium weight.
Each letter has generous breathing room between it."
```

## Font DNA Block (proven)
```
"BODY FONT: Premium geometric sans-serif — the level of Aesop packaging, Kinfolk magazine,
Apple product pages. ALL body copy and info text MUST be set in ALL CAPS with VERY WIDE
letter-spacing — this is the #1 rule for making type look expensive. Every letter has
breathing room. Think luxury brand packaging: S P A C E D   O U T   C A P S.
Medium weight (not light, not bold — MEDIUM)."
```

## Why ALL CAPS works for NANO:
1. Caps are simpler letterforms — NANO renders them more consistently
2. Wide tracking masks imperfect kerning (NANO's weakness)
3. ALL CAPS + wide spacing = universal luxury signal (Aesop, Apple, Kinfolk, Cereal Magazine)
4. Medium weight avoids thin/wispy (looks weak) or bold/chunky (looks cheap)

## Info block format (proven)
```
"[DATE] · [TIME] · [AUDIENCE]"
"[PRICE TIER 1] | [PRICE TIER 2]"
"[VENUE] · [CTA]"
```
ALL CAPS, centered, dot/pipe separators, generous line spacing.

## Still won't match real fonts
This is NANO's ceiling for typography. For pixel-perfect Mabry Pro, would need PIL or Recraft V4. User chose NANO all the way — this is the best we can get.
