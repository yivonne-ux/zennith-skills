## SOCIAL MEDIA PRODUCTION — UNIVERSAL LEARNINGS

### 1. COPY, NEVER IMAGINE
- ALL post copy must come from REAL viral posts with proven engagement
- Never write quotes/hooks from imagination
- Scrape IG accounts, Pinterest, TikTok for actual trending text
- The viral ref provides BOTH the format AND the copy
- Your job: change the ART STYLE to the brand, not the words

### 2. VIRAL REFERENCE FIRST — ALWAYS
- Never generate content from scratch
- Find the viral/trending post FIRST, then adapt to brand
- Even for illustrations: the viral ref = the FORMAT, style ref = the ART
- Describe the viral ref's layout in TEXT prompt, use style ref as Image 1
- "It's OK if the ref looks not nice — we can adapt brand vibes and aesthetic"

### 3. COLOR GRADING = MULTI-LAYER, NEVER FLAT
- NEVER use flat single-color overlay ("shift to pink")
- Always specify: shadow tone, midtone, highlight tone SEPARATELY
- Film-like tonal separation — shadows lean one direction, highlights another
- Specify temperature, grain placement, vignette
- "When you change the image colour, the editing has to have multiple layers of tone"

### 4. EVERY POST MUST HAVE A PURPOSE
- Would someone DM this to 5 friends? If no → reject
- Pretty photo + generic poetic copy = REJECT ("comfort in a box" = no value)
- Every post must deliver: viral humor, genuine education, emotional trigger, or identity claim
- "No meaning, no viral factor, not a value to customer" = reject

### 5. FOOD EDUCATION POSTS NEED GENUINE VALUE
- Must teach something SPECIFIC and SURPRISING
- Use proven viral formats: "Same calories different results", "THIS OR THAT", "I eat a little vs I eat a lot"
- The brand must GENUINELY WIN the comparison — not tie, not be equivalent
- Volume comparison works: "1 regular meal = 2 brand meals" is shocking and shareable
- Never make false claims — if you don't know the actual data, don't state it

### 6. SACRED PHOTOGRAPHS — NEVER AI-GENERATE PRODUCTS
- Real product/food photos are UNTOUCHABLE
- Place EXACTLY as provided — same container, same plating, same colors
- If the copy references a specific product, the photo MUST BE that product
- Image-copy mismatch = instant reject ("nasi lemak" text + golden rice photo = WRONG)

### 7. LABEL ACCURACY
- Every label/pointer must point to the CORRECT item
- Audit label routing before presenting
- Duplicate labels on wrong items = defect, not artistic choice

### 8. WARMTH IS DEFAULT (brand-specific warmth, not generic bright)
- Default warmth definition depends on the BRAND, not a universal rule
- Some brands = bright sunlit (wellness). Some brands = candlelit sparkle (Mirra). Check brand DNA.
- "Too dark" only applies if there's no SPARKLE or TEXTURE to compensate
- Dark + sparkle/glitter CAN be on-brand (check brand file)
- FLAT/DULL is always wrong. But flat ≠ dark. Flat = no texture, no depth, no atmosphere
- The real rule: never CLINICAL or FLAT. Always ATMOSPHERIC and TEXTURED.

### 9. REFERENCE QUALITY DETERMINES OUTPUT QUALITY
- Weak/plain references = weak outputs. No amount of prompting saves a weak ref
- The ref itself must be PREMIUM and INTERESTING
- A blank card is not a reference. A plain notepad is not a reference.
- Scrape across platforms (IG, Pinterest, TikTok, web) for the HIGHEST engagement refs

### 10. BACKGROUND MUST BE ON-BRAND
- Background is NOT neutral — it's a brand signal
- Nature/green/sage = off-brand for pink brands
- Always check: "does this background FEEL like the brand?"

### 11. FORMAT INTEGRITY
- Labeled photo meme = must be REAL PHOTO with labels, not illustration
- Typography = must match VIRAL typography aesthetics, not generic clean design
- Illustration = must follow locked style (Korean webtoon for Mirra, etc.)
- Copyright-risky characters (Disney, Pixar) = NEVER use

### 12. SINGLE LOGO RULE
- Either PIL logo OR NANO-rendered brand text, NEVER both
- Logo must NEVER overlap rendered text
- Place logo in cleanest zone AWAY from text content

### 13. TEXT INTEGRITY GATE
- All text must render correctly — no garbled characters
- No residual text from reference images (other brand names, watermarks)
- No typos
- CN characters must render correctly for Chinese posts

### 14. DON'T PIXEL-FILL TO FIX — REGENERATE
- Numpy pixel-fill to remove logos leaves grey artifacts
- If something needs removing, regenerate the whole image
- Quick fixes create visible defects

### 15. PRODUCT POSTS vs VIRAL POSTS
- "Your week sorted" with 5 product photos = AD
- Same products in a VIRAL comparison format = ORGANIC content
- The difference is whether the FORMAT comes from proven viral or from product marketing
- Feed brand context INTO viral formats, don't build formats around products
