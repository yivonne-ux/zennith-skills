## Andromeda Format Diversity Strategy

### Core Principle
Each VISUALLY UNIQUE creative gets its own Andromeda Entity ID → finds a DIFFERENT audience pocket. Format diversity > format perfection. A "good enough" bingo card finding a RM30 CPA audience is worth more than a perfect receipt fighting the same audience as TNG-PROMO.

**Why:** Vertical scaling (more budget same creative) causes frequency death → CPA rises. Horizontal scaling (more FORMATS same budget) opens new audience pools → CPA stays low.

**How to apply:** Every production batch must introduce NEW visual format families, not variations of existing ones. Target 15-20 unique Entity IDs per campaign.

### Post Type Taxonomy (40+ archetypes identified)
Not limited to "ad formats." ANY viral post type from ANY category (fashion, tech, lifestyle, humor) can be adapted to food/brand context. The FORMAT is the hook, not the food.

**Proven for Pinxin (batch 1-3):**
News headline, meme ("what's on my mind"), pain/comparison, bold typography, editorial red/green flags, scrapbook/torn paper, tier list, magazine cover, mixed-media collage, fork-in-road illustration, comic strip, Spotify wrapped, abundance grid, gamification/quiz, before/after split, notes app, authority carousel, bingo card, what-I-eat meal plan, ugly/lo-fi, price breakdown, "send this to..." share trigger, podcast quote clip, emotional family stat, Chinese breaking news, UGC testimonial, utility guide
