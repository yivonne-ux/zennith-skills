## Budget gates (hard rules)
- Scale increase: ROAS >3.5x for 5 consecutive days
- Kill ad: cost/WA >RM12 after 7 days AND spend >RM200
- Kill concept: ALL ads in concept >RM10/WA
- Frequency alert: any ad set >2.0
- ROAS drop >30% after increase: revert immediately
- Scale ad: cost/WA <RM7 sustained 7 days → +20% every 48h via Post ID
