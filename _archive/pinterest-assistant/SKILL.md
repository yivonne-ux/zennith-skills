# pinterest-assistant

Pinterest is often login-walled / anti-bot. This skill uses a **human-in-the-loop** workflow:
- you share **pin links / board links / screenshots**
- zenni extracts the vibe and returns **Style DNA + prompt packs + do/don't**

## Inputs (pick any)
- 1–5 pin links (best)
- a board link + tell me which pins to focus (or send screenshots)
- screenshots of the reference images

## Output (standard)
1) **Style DNA**
   - palette (5–8 colors + words)
   - brush/texture
   - lighting
   - composition
   - typography vibe
2) **Mood words** (3–7)
3) **Prompt pack**
   - NanoBanana prompt (image generator)
   - Midjourney prompt + suggested params
4) **Do/Don't**
5) **3 variations** (more premium / more playful / more conversion)

## How to ask
- "zenni extract vibe" + paste link(s)
- "zenni make prompt pack" + attach screenshot(s)

## Notes
- If the link is blocked, screenshots always work.
