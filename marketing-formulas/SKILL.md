## 1. COPYWRITING FRAMEWORKS (32 Formulas)

### Tier 1 — The Essentials (Use Daily)

**AIDA** — Attention → Interest → Desire → Action
- Best for: Landing pages, email, product ads
- Example: "🔥 Malaysia's #1 vegan rendang (A) → Made from 12 traditional spices (I) → Imagine restaurant-quality meals in 15 min (D) → Shop now, RM15 off first order (A)"

**PAS** — Problem → Agitation → Solution
- Best for: Pain-point ads, retargeting, email
- Example: "Tired of tasteless vegan food? (P) → Every meal feels like a compromise, your family complains... (A) → Pinxin Poon Choi — the dish that makes everyone forget it's vegan (S)"

**BAB** — Before → After → Bridge
- Best for: Transformation content, testimonials, UGC
- Example: "Before: I spent 2 hours cooking healthy meals (B) → After: 15-minute gourmet vegan dinners every night (A) → Bridge: GAIA Eats meal kits changed everything (B)"

**ACCA** — Awareness → Comprehension → Conviction → Action
- Best for: Educational funnels, complex products
- Funnel mapping: TOFU(A) → MOFU(C+C) → BOFU(A)

**4Ps** — Promise → Picture → Proof → Push
- Best for: Sales pages, long-form ads
- Example: "Promise: Lose 5kg in 30 days eating delicious food → Picture: Imagine stepping on the scale and smiling → Proof: 2,847 Malaysians already did it → Push: Start your transformation today"

**FAB** — Features → Advantages → Benefits
- Best for: Product descriptions, comparison ads
- Example: "100% plant-based ingredients (F) → No cholesterol, low sodium (A) → Feel energized all day without the guilt (B)"

### Tier 2 — Advanced Frameworks

**PASTOR** — Problem → Amplify → Story → Transformation → Offer → Response
- Best for: Long-form sales letters, webinar scripts
- The "preacher" formula — builds emotional momentum

**StoryBrand (Donald Miller)** — Character → Problem → Guide → Plan → CTA → Success → Failure
- Best for: Brand messaging, website copy, about pages
- Key insight: Customer is the HERO, brand is the GUIDE

**QUEST** — Qualify → Understand → Educate → Stimulate → Transition
- Best for: Email sequences, webinar funnels

**SLAP** — Stop → Look → Act → Purchase
- Best for: Scroll-stopping social ads

**OATH** — Oblivious → Apathetic → Thinking → Hurting
- Audience awareness spectrum — match copy to where they are

**Eugene Schwartz 5 Levels of Awareness:**
1. Unaware → Use curiosity, don't mention product
2. Problem-aware → Agitate the pain
3. Solution-aware → Show your solution is best
4. Product-aware → Handle objections, build trust
5. Most aware → Just give the offer

**APP** — Agree → Promise → Preview
- Best for: Blog intros, video intros, email openers

**SSS** — Star → Story → Solution
- Best for: Case studies, testimonial ads

**PPPP** — Picture → Promise → Prove → Push
- Best for: Social media ads, carousel posts

**The 1-2-3-4 Formula:**
1. What I've got for you
2. What it's going to do for you
3. Who am I
4. What you need to do next

**So What? Formula** — State fact → "So what?" → Answer the benefit
- Forces benefit-focused writing

**The Approach Formula** — Arrive → Problem → Propose → Reasons → Objections → Act → Close → Hook-to-next
- Best for: Long sales letters

### Tier 3 — Specialist Formulas

**FOREST** — Facts → Opinions → Reasons → Examples → Stats → Testimonials
- Best for: Authority-building content

**FACE** — Familiar → Audience → Cost → Education
- Best for: B2B content marketing

**AIDPPC** — Attention → Interest → Description → Persuasion → Proof → Close
- Extended AIDA for complex sales

**Ray Edwards' PASOP** — Problem → Amplify → Solution → Objection → Proof
- Best for: Infomercial-style ads

**Dan Kennedy's 3-Step:** Interrupt → Engage → Educate
- Best for: Direct response advertising

**Gary Halbert's SRDB:** Stop → Read → Desire → Buy
- Best for: Headlines and subject lines

**Joe Sugarman's Slippery Slide:**
- Every element exists to get you to read the NEXT line
- First sentence should be short and compelling
- Best for: Long-form ads, emails

**Bob Bly's BFD:** Beliefs → Feelings → Desires
- Best for: Understanding audience before writing

**Perry Belcher's 21-Step Sales Letter:**
1. Call out audience → 2. Get attention → 3. Backup the big promise → 4. Create intrigue → 5. Unique mechanism → ... → 21. Close with urgency

**Robert Collier's Conversational Formula:**
- Write as if talking to one person
- Enter the conversation already in their head
- Best for: Email, DMs, WhatsApp marketing

**Victor Schwab's AIDCA:** Attention → Interest → Desire → Conviction → Action
- Added Conviction step — proof before action

**Frank Dignan's AAPPA:** Attention → Advantage → Proof → Persuasion → Action

**Henry Hoke's AICPBSAWN:** Attention → Interest → Credibility → Prove → Benefits → Scarcity → Action → Warn → Now
- The "kitchen sink" formula — everything in one

## 2. PERSUASION PSYCHOLOGY

### Cialdini's 6 Principles (+ 7th)
1. **Reciprocity** — Give value first, they feel obligated to return
2. **Commitment/Consistency** — Small yes → big yes (foot-in-door)
3. **Social Proof** — "10,000 Malaysians chose us" / reviews / UGC
4. **Authority** — Expert endorsement, certifications, media mentions
5. **Liking** — Be relatable, attractive, similar to audience
6. **Scarcity** — Limited time, limited stock, exclusive access
7. **Unity** — "We're the same tribe" (shared identity)

### Behavioral Economics Triggers
- **Loss Aversion** — "Don't miss out" > "Get this" (2x more powerful)
- **Anchoring** — Show high price first, then discounted price
- **Decoy Effect** — Add a worse option to make target option look better
- **Endowment Effect** — Free trial, "it's already yours"
- **Bandwagon Effect** — "Trending", "Best-seller", "Most popular"
- **Paradox of Choice** — Fewer options = more conversions (3 tiers max)
- **Peak-End Rule** — People remember the peak moment and the ending
- **Zeigarnik Effect** — Incomplete = memorable (use cliffhangers)
- **IKEA Effect** — Let them customize/build = higher perceived value
- **Mere Exposure** — Retargeting works because familiarity breeds liking

### Emotional Triggers (Top 20 for Ads)
1. Fear of missing out (FOMO)
2. Belonging / community
3. Guilt (health, environment)
4. Pride / status
5. Curiosity / mystery
6. Nostalgia (mak's cooking)
7. Hope / aspiration
8. Anger / frustration (at problem)
9. Relief / comfort
10. Surprise / delight
11. Trust / safety
12. Excitement / anticipation
13. Envy (they have it, you don't)
14. Gratitude
15. Empathy / compassion
16. Achievement / progress
17. Freedom / independence
18. Love / care (for family)
19. Humor / joy
20. Urgency / time pressure

## 3. SOFT SELL TECHNIQUES

### The Value-First Framework
```
Step 1: Give 90% value, 10% pitch
Step 2: Educate → Entertain → THEN sell
Step 3: Let the product be discovered, not pushed
```

### Soft Sell Methods
1. **Education-Based Selling** — Teach something useful, product is the tool
   - "5 ways to eat healthier this month" → Product appears in tip #3
2. **Story-Based Selling** — Share a journey, product is part of the story
   - "My 30-day plant-based challenge" → naturally features products
3. **Community-Based Selling** — Build tribe first, sell to members
   - Facebook group, WhatsApp community → exclusive offers
4. **Social Proof Selling** — Let others do the selling
   - UGC, reviews, testimonials, influencer content
5. **Content Marketing** — Blog/video SEO → organic discovery → purchase
6. **Native Advertising** — Ads that look like content (LoFi, UGC-style)
7. **Seeding** — Give product to influencers, let them post naturally
8. **Behind-the-Scenes** — Show the making process, build emotional connection
9. **Value Ladder** — Free → cheap → mid → premium (upsell naturally)
10. **Trojan Horse** — Viral content with subtle brand integration

### Hard Sell vs Soft Sell — When to Use
| Funnel Stage | Sell Type | Ratio |
|---|---|---|
| TOFU | 100% soft | Educate + entertain only |
| MOFU | 70% soft / 30% hard | Value + subtle product mentions |
| BOFU | 30% soft / 70% hard | Direct offer with proof |
| Retarget | 20% soft / 80% hard | They already know you — close |

## 5. UPSELL / DOWNSELL / CROSS-SELL

### The Value Ladder
```
FREE        → Lead magnet (recipe ebook, meal plan)
RM 15-30    → Entry product (single item, trial pack)
RM 50-100   → Core product (bundle, variety pack)
RM 150-300  → Premium (subscription, mega bundle)
RM 500+     → VIP (catering, corporate, wholesale)
```

### Upsell Frameworks
1. **Bundle Upsell** — "Add [complementary product] for 20% off"
2. **Size Upsell** — "Get the family pack for only RM15 more"
3. **Subscription Upsell** — "Subscribe & save 15% every month"
4. **Premium Upsell** — "Upgrade to organic/premium version"
5. **Speed Upsell** — "Express delivery for RM5 more"

### Downsell Frameworks (Cart Abandonment)
1. **Smaller Commitment** — "Not ready for the bundle? Try a single pack"
2. **Payment Plan** — "Split into 3 payments of RM33"
3. **Trial Offer** — "Try for RM9.90 first, full price later"
4. **Stripped Version** — "Get the basic kit without [premium feature]"
5. **Time Extension** — "Take 48 more hours, here's 10% off"

### Cross-Sell Frameworks
1. **Complementary** — "People who bought rendang also love our sambal"
2. **Complete the Set** — "You're missing the Poon Choi to complete your CNY feast"
3. **Frequently Bought Together** — Amazon-style bundles
4. **Recipe Bundle** — "Get all ingredients for this recipe in one click"
5. **Seasonal Cross-Sell** — "Add our CNY gift box to your order"

### Post-Purchase Upsell Sequence
```
Day 0:  Thank you email + how-to-use content (soft)
Day 3:  "How's your [product]?" + recipe ideas (soft)
Day 7:  Cross-sell complementary product (medium)
Day 14: Subscription offer — "Never run out" (medium)
Day 21: Loyalty program invitation (soft)
Day 30: Replenishment reminder + bundle offer (hard)
Day 45: Win-back if no repurchase — discount code (hard)
```

## 7. PLATFORM-SPECIFIC FORMULAS

### Meta (Facebook + Instagram)
- **Feed:** 1:1 or 4:5, first 125 chars visible, use emoji, 10-15 hashtags (IG)
- **Stories:** 9:16, 5-15s, sticker CTA, poll for engagement
- **Reels:** 9:16, 15-90s, trending audio boost, text overlay essential
- **Best formula:** PAS for feed, Hook→Value→CTA for Reels
- **Winning patterns:** UGC outperforms polished, faces > graphics, text overlay > clean

### TikTok
- **Format:** 9:16, 15-60s sweet spot, trending sounds = algorithm boost
- **Best formula:** Hook (3s) → Value (15-40s) → CTA (3s)
- **Winning patterns:** LoFi > HiFi, authentic > polished, controversy = reach, series = followers

### YouTube Shorts
- **Format:** 9:16, 15-60s, thumbnail less important (autoplay)
- **Best formula:** Curiosity hook → Fast delivery → Surprise ending
- **Winning patterns:** Tutorial style, satisfying process videos, transformation

### Email (Klaviyo)
- **Subject line:** 30-50 chars, emoji optional, personalize {first_name}
- **Best formula:** PAS or BAB for promotional, AIDA for launch
- **Winning patterns:** Story-based > promotional, 1 CTA > multiple, plain text > designed (for certain segments)

### WhatsApp
- **Format:** Short, conversational, emoji-heavy, voice notes for personal touch
- **Best formula:** Direct offer with social proof: "Hi {name}! 🌿 2,000+ customers love our [product]. Special offer just for you..."
- **Winning patterns:** Personalized > broadcast, image+text > text-only, reply incentive

### Landing Page
- **Above fold:** Headline (benefit) + subhead (how) + hero image + CTA
- **Best formula:** AIDA structure, StoryBrand messaging
- **Winning patterns:** 1 CTA per page, social proof near CTA, mobile-first, <3s load time
- **A/B test priority:** Headline > CTA > hero image > social proof > layout

### Shopee/Lazada
- **Title:** Keyword-stuffed but readable, 120 chars max
- **Images:** 1:1, white background for main, lifestyle for secondary
- **Description:** FAB formula, bullet points, specs table
- **Winning patterns:** Bundle pricing, free shipping threshold, flash deals

## 9. GROWTH HACKING FRAMEWORKS

### Viral Loop Mechanics
1. **Referral Loop** — Refer a friend, both get RM10 off
2. **Content Loop** — User creates content → attracts new users → they create content
3. **Social Loop** — Share order/review on social → get discount code
4. **Gamification Loop** — Points/badges/streaks → status → sharing
5. **Community Loop** — Join WhatsApp group → exclusive content → invite friends

### Growth Experiments Framework (ICE Score)
| Experiment | Impact (1-10) | Confidence (1-10) | Ease (1-10) | ICE Score |
|---|---|---|---|---|
| [experiment name] | ? | ? | ? | = I×C×E/10 |

Priority: Run highest ICE score first. Kill experiments at 2 weeks if no signal.

### The AARRR Pirate Metrics
1. **Acquisition** — How do they find you? (Ads, SEO, social, referral)
2. **Activation** — First "aha" moment (first purchase, email signup)
3. **Retention** — Do they come back? (Repeat purchase, subscription)
4. **Revenue** — Do they pay? (AOV, LTV, ROAS)
5. **Referral** — Do they tell others? (NPS, referral rate, UGC)

### Winner Detection → Duplication → Scale
```
DETECT:
  - Creative with CTR > 2x average → FLAG as potential winner
  - Run for 48-72 hours with statistical significance test
  - If p < 0.05 and lift > 20% → CONFIRMED WINNER

DUPLICATE:
  - Take winning creative, generate 10 variants:
    * Same hook, different visual
    * Same visual, different hook
    * Same message, different format (image→video, video→carousel)
    * Same creative, different audience
    * Same creative, different placement

SCALE:
  - Day 1-3: RM50/day (test budget)
  - Day 4-7: If ROAS > target → RM150/day
  - Day 8-14: If ROAS holds → RM500/day
  - Day 15+: If ROAS holds → uncap (with daily monitoring)
  - KILL: If ROAS drops >30% for 3 consecutive days → pause
```

### Pricing Psychology
- **Charm Pricing** — RM29.90 not RM30 (9-ending = perceived deal)
- **Anchor Pricing** — Show RM99 crossed out, RM59 highlighted
- **Bundle Pricing** — 3 for RM79 (vs RM35 each = RM105)
- **Decoy Pricing** — Small RM25, Medium RM45, Large RM49 (large = obvious choice)
- **Subscribe & Save** — 15% off recurring (LTV play)
- **Free Shipping Threshold** — "Free shipping over RM80" (increases AOV)
- **Limited Edition Premium** — 20% markup for seasonal/exclusive
- **Pay-What-You-Can** — For community building (rare, brand play)

## 11. LANDING PAGE OPTIMIZATION

### Above-The-Fold Formula
```
┌────────────────────────────────────────┐
│  HEADLINE: [Benefit-driven, 8-12 words]│
│  SUBHEAD: [How you deliver the benefit]│
│                                        │
│  [HERO IMAGE or VIDEO]                 │
│                                        │
│  [CTA BUTTON — contrasting color]      │
│  "Start Your Free Trial" > "Submit"    │
│                                        │
│  [Social proof: "Join 10,000+ ..." ]   │
└────────────────────────────────────────┘
```

### A/B Test Priority (Impact Order)
1. **Headline** — Biggest impact on conversion (test 5+ variants)
2. **CTA text + color** — "Get Started Free" > "Sign Up"
3. **Hero image/video** — Faces > illustrations > abstract
4. **Social proof placement** — Near CTA > header > footer
5. **Price presentation** — Anchored > plain > hidden
6. **Form length** — Fewer fields > more fields (usually)
7. **Page length** — Depends on awareness level (low aware = long, high aware = short)

### Conversion Rate Benchmarks (E-commerce Malaysia)
- Landing page: 2-5% (good), 5-10% (great), 10%+ (exceptional)
- Add to cart: 8-15%
- Cart to purchase: 30-50%
- Email signup: 3-8%
- Shopee/Lazada: 1-3% (marketplace = lower intent per visit, higher volume)
